Вентиляторы и теплоотвод

Описание функций

На странице Вентиляторы и теплоотвод можно выполнить следующие операции:

  • Просмотр журнала данных по температуре поступающего воздуха сервера.
  • Просмотр и настройка параметров для регулирования работы вентиляторов.

    Все параметры на странице Настройки умного охлаждения вступают в силу только при изменении режима регулирования скорости вентиляторов с Ручной на Авто.

  • Просмотр информации о вентиляторах, установленных в сервере.

Описание параметров

Табл. 1 Температура поступающего воздуха

Параметр

Описание

Температура поступающего воздуха

Выборка данных по температуре поступающего воздуха, осуществляемая каждые 10 минут в течение последней недели.

Текущее значение

Последнее значение температуры, измеренное датчиком на входе.

Удалить

Кнопка Удалить предназначена для удаления всех данных.

Обновить

Кнопка Обновить предназначена для обновления текущих статистических данных.
Табл. 2 Данные о вентиляторах

Параметр

Описание

Основная информация

Основная информация о модулях вентиляторов сервера, включая номер слота, имя, модель, скорость и соотношение скорости каждого модуля вентиляторов.

Обновление в реальном времени

Включение или отключение функции обновления информации о вентиляторе в реальном времени.

После включения данной функции параметры Скорость и Соотношение скорости автоматически обновляются каждую секунду.

По умолчанию данная функция отключена.

ПРИМЕЧАНИЕ:

Если вы обновите текущую страницу, переключитесь на другую страницу или закроете страницу после включения функции Обновление в реальном времени, эта функция будет автоматически отключена.
Табл. 3 Настройки умного охлаждения

Параметр

Описание

Режим экономии энергии

Это режим по умолчанию для системы воздушного охлаждения.

Настройка работы вентиляторов в соответствии с нагрузкой системы для лучшего теплоотвода и экономии электроэнергии.

Тихий режим

Работа вентиляторов на пониженной скорости для тихой работы системы теплоотвода.

Высокопроизводительный режим

Работа вентиляторов на максимальной скорости для оптимального охлаждения компонентов и работы системы.

Пользовательский режим
Предоставляет пользователям интерфейсы для настройки следующих параметров. В зависимости от нагрузки на сервер и состояния теплоотвода, допустимые диапазоны значений предоставляются как справочные данные.
  • Целевая температура ЦПУ
  • Целевая температура исходящего воздуха
  • Целевая температура модуля памяти DIMM
  • Скорость вентиляторов
ПРИМЕЧАНИЕ:
  • Скорость вентилятора, соответствующая более высокому температурному диапазону, должна быть больше скорости вентилятора, соответствующей более низкому температурному диапазону.
  • Параметры, доступные в пользовательском режиме, зависят от модели сервера.
  • Если какая-либо фактическая температура выше целевой температуры настройки скорости вентилятора, BMC увеличивает скорость вентилятора для снижения температуры. Если все фактические температуры ниже целевой температуры настройки скорости вентилятора, BMC регулирует скорость вентилятора на основе значения Скорость вентиляторов.
  • Если после замены ЦПУ текущее значение параметра Целевая температура ЦПУ превышает максимальную рабочую температуру нового ЦПУ, то BMC автоматически установит параметру Целевая температура ЦПУ значение, соответствующее максимальной температуре, допустимой для нового ЦПУ.

Настройки умного охлаждения

Например, чтобы установить Пользовательский режим умного охлаждения, выполните следующие действия:

Некорректные настройки пользовательского режима могут стать причиной недостаточного теплоотвода.

  1. Нажмите Настройки.
  2. Выберите Пользовательский режим.
  3. Настройте параметры Целевая температура ЦПУ и Целевая температура исходящего воздуха в зависимости от отображаемого на экране диапазона значений.
  4. Настройте скорость вращения вентиляторов в соответствии с диапазоном температур поступающего воздуха.
  5. Нажмите Сохранить.

    При успешной настройке режима умного охлаждения на экране появится сообщение «Операция выполнена успешно».
