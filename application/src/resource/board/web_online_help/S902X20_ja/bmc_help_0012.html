ファンと放熱

機能説明

ファンと放熱画面で、以下の操作を実行できます。

  • ファン回転速度の調整モードの表示および設定を行います。

    ファン回転速度の調整モードが[手動]の場合、ファン回転速度の調整モードが[自動]に変更された場合にのみ、[スマート冷却設定]画面のすべての設定が有効になります。

  • サーバーに取り付けられたファンの情報を表示します。

カテゴリ

吸気口温度

表 2 ファン情報

カテゴリ

説明

基本情報

各ファンモジュールのスロット番号、名称、型番、速度、速度比など、部品番号、サーバーのファンモジュールに関する基本情報。

リアルタイム更新

ファン情報のリアルタイム更新を有効または無効にします。

有効にすると、ファン回転速度と速度比は1秒ごとに自動的に更新されます。

デフォルトでは無効になっています。
表 3 スマート冷却設定

カテゴリ

説明

省エネルギーモード

これは、空冷システムのデフォルトモードです。

システム負荷に合わせてファンの回転速度を調整し、放熱性と省電力性を向上させます。

静音モード

ファンの回転速度を最小化し、静かに冷却します。

ハイパフォーマンスモード

重要なポーネントの冷却とシステム実行のために、ファンの回転速度を上げます。

カスタムモード
カスタマムインターフェースを提供し、ユーザーが以下のパラメータを設定できるようにします。参考としてサーバーの負荷や放熱の状況に応じてオプションの値の範囲が示されます。
  • CPUターゲット温度
  • ファンの回転速度
NOTE:
  • 高温範囲に対応する回転速度は、低温範囲に対応する回転速度より大きくする必要があります。
  • カスタムモードで使用できるパラメータは、サーバーモデルによって異なります。
  • CPUを交換する際に新しいCPUの最高動作温度が現在の[CPUターゲット温度]の値より低い場合、BMCは自動的に[CPUターゲット温度]の値を新しいCPUが許容する最高温度に変更します。

スマート冷却設定

以下は、[カスタムモード]を例としてスマート冷却設定の操作方法について説明します。

カスタムモードが正しく設定されていない場合、放熱が不足になる可能性があるため、慎重に選択してください。

  1. [設定]をクリックします。
  2. [カスタムモード]を選択します。
  3. 表示された値の範囲に基づいて、[排気口ターゲット温度]を設定します。
  4. 吸気口の温度範囲に対応するファン速度を設定します。
  5. [保存]をクリックします。

    操作成功のメッセージが表示されると、スマート冷却設定に成功したことを示します。
