オンラインユーザー

機能概要

[オンラインユーザー]画面で、次の操作を実行できます。

  • BMCにログインしたユーザーに関する情報を照会します。
  • オンラインユーザーを強制的にログアウトします。

管理者のみが強制的にログアウトすることができます。

パラメータの説明

表 1 オンラインユーザー

パラメータ

説明

ユーザー名

BMCまたはKVMにログインしたユーザーの名前。

ログイン方法

ユーザーのログイン方法。

次のオプションがあります。

  • GUI(SSO):ユーザーがSSO経由でBMC WebUIにログインします。
  • GUI:ユーザーがSSO経由せずにBMC WebUIにログインします。
  • CLI:ユーザーがコマンドラインインターフェース（CLI）を介してBMCにログインします。
  • KVM:ユーザーがリモート仮想コンソールを介してOSにログインします。
  • Redfish:ユーザーがRedfishインターフェースを介してBMCにログインします。
  • VNC:ユーザーはVirtual Network Console（VNC）を介してオペレーティングシステム（OS）にログインします。

IPアドレス

BMC.に接続してログインするためのIPアドレス。

値:IPアドレスまたは[COM]

NOTE:

[COM]は、ユーザーがシリアルポートを介してBMCにログインすることを示します。

ログイン時間

ユーザーがBMCにログインした時間。

操作

ユーザーを強制的にログアウトします。

[ログアウト]をクリックしてユーザーを強制的にログアウトします。
