Smart Provisioning管理

機能説明

Smart Provisioningはインテリジェントなサーバー展開ツールです。

[Smart Provisioning管理]画面でサーバー情報収集の機能を有効または無効にすることができます。

パラメータの説明

表 1 Smart Provisioning管理

パラメータ

説明

サーバー情報の収集

サーバー情報収集の機能を設定します。

この機能を有効にした場合、PSUの電源投入時にインバンドシステムが初めて起動した際、インバンドシステムがSmarting Provisioningにアクセスしてサーバー情報を収集します。その後、サーバーは[BIOS構成]の[優先起動媒体]および[起動順位]に従って起動します。

デフォルトでは無効になっています。
