BMC管理ソフトウェア概要

インテリジェントベースボード管理コントローラ（BMC）は、豊富な管理機能と機能を提供します。

  • 豊富な管理インターフェース

    BMCは以下のインターフェースをサポートしています。

    • Data Center Manageability Interface（DCMI）仕様1.5
    • Intelligent Platform Management Interface (IPMI)1.5/2.0
    • コマンドラインインターフェース（CLI、Command-line interface）
    • Redfishインターフェース
    • Hypertext Transfer Protocol Secure（HTTPS）インターフェース
    • Simple Network Management Protocol（SNMP）インターフェース
  • 障害監視と診断

    次の機能により、BMCは潜在リスクを検出し、7*24時間体制で信頼性の高いシステム運用を保証します。

    • システムクラッシュ直前のスクリーンショットと録画再生

      システムクラッシュの原因解析に役立ちます。

    • スクリーンのスナップショットと録画

      定期的な予防保守または操作の記録と監査を簡素化します。

    • 障害診断と管理（FDM、Fault diagnosis & management）機能

      コンポーネントに対する正確な故障診断により、故障したコンポーネントの特定と交換が容易になります。

    • Syslog、Trap、Emailによるアラームレポート

      上位のネットワーク管理システム（NMS、network management system）へのサーバーアラーム情報の収集が便利になります。

  • セキュリティ管理
    • BMCはイメージミラーリングを使用してシステムのセキュリティを向上させます。実行中のソフトウェアが故障しても、システムはバックアップイメージから起動できます。
    • 多様なユーザーセキュリティ制御インターフェースにより、ログインセキュリティが保証されます。
    • データ転送のセキュリティを確保するために、多種類の証明書のインポートと交換をサポートしています。
  • システムメンテナンスインターフェース
    • BMCはキーボード、ビデオ、マウス(KVM)、仮想メディアをサポートし、リモートメンテナンスを容易にします。
    • RAIDのアウトオブバンド監視と設定をサポートし、RAID構成の効率性と管理性を向上させます。
    • Smart Provisioning機能により、DVDを使用することなくOSのインストール、RAID構成、およびアップグレードを実装し、サーバーのインストールと構成を簡素化します。
  • 多様なネットワークプロトコル
  • BMCはネットワークタイムプロトコル（NTP、Network Time Protocol）をサポートし、時間の設定を容易にし、時刻同期を確保します。
  • BMCはドメイン管理とディレクトリサービスをサポートし、ネットワーク管理を簡素化します。
  • インテリジェントな電源管理
    • パワーキャッピングにより、展開密度を向上します。
    • 動的な省エネにより、運用コストを削減します。
  • ライセンス管理

    ライセンス管理により、承認されたユーザーがBMCのアドバンスド特性を使用できるようになります。

    アドバンスドエディションは標準エディションよりもっと多くのアドバンスド特性を提供します。例えば:
    • RedfishによってOSをデプロイします。
    • Enable Kunpeng acceleration engines, including the hardware Security Engine (SEC), High Performance RSA Engine (HPRE), RAID DIF Engine (RDE), and ZIP engine.
