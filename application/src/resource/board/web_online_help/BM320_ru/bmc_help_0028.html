Пользователи в сети

Описание функций

На странице Пользователи в сети доступны следующие операции:

  • Просмотр пользователей, которые выполнили вход в BMC.
  • Принудительный выход пользователей из системы.

Только администратор может завершить сеанс пользователя.

Описание параметров

Табл. 1 Пользователи в сети

Параметр

Описание

№

Порядковый номер вошедшего в систему пользователя.

Имя пользователя

Имя пользователя, вошедшего в BMC или KVM.

Способ входа

Режим, в котором пользователь выполнил вход.

Доступные опции:

  • Графический интерфейс пользователя (GUI)(SSO): пользователь выполнил вход в WebUI BMC через SSO.
  • Графический интерфейс пользователя (GUI): пользователь выполнил вход в WebUI BMC без использования SSO.
  • CLI: пользователь выполнил вход в BMC через интерфейс командной строки (CLI, Command-Line Interface).
  • KVM: пользователь выполнил вход в ОС через удаленную виртуальную консоль.
  • Redfish: пользователь выполнил вход в BMC через интерфейс Redfish.
  • VNC: пользователь выполнил вход в ОС через виртуальную сетевую консоль.

IP-адрес

IP-адрес для подключения и входа в BMC.

Значения: IP-адрес или COM

ПРИМЕЧАНИЕ:

Значение COM означает, что пользователь выполнил вход в BMC через последовательный порт.

Время входа

Время, когда пользователь выполнил вход в BMC.

Операция

Принудительный вывод пользователя.

Нажмите Выход для завершения сеанса пользователя.
