Управление устройством USB

Описание функций

На странице Управление устройством USB можно запросить и настроить порт управления прямого подключения BMC.

Функция управления устройством USB доступна при использовании на сервере порта управления прямого подключения BMC.

Описание

Управление устройством USB
После включения данной функции вы можете получить доступ к BMC через USB-порт на монтажной проушине сервера одним из следующих способов:
  • Локальный ПК: вы можете получить доступ к BMC WebUI (IP-адрес 169.254.1.5) с помощью браузера или получить доступ к BMC CLI с помощью SSH при использовании локального ПК.
  • Мобильное устройство: вы можете использовать мобильное приложение SmartServer для получения доступа к BMC на мобильном устройстве.
ПРИМЕЧАНИЕ:
  • Данная функция включена по умолчанию.
  • Данная функция поддерживается только ПК, работающим под управлением Windows 10, мобильным телефоном, работающим под управлением Android.

Состояние устройства USB

Состояние устройства USB.

  • Подключено: подключение к устройству осуществляется через USB-порт.
  • Не подключено: не осуществляется подключение к устройству через USB-порт.
