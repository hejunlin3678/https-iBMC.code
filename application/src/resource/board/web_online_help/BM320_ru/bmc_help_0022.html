Рабочие записи

Описание функций

На странице Рабочие записи можно создавать и просматривать рабочие записи.

  • Максимальная длина рабочей записи составляет 255 символов. Пользователь может создать максимум 20 рабочих записей. Если количество записей превышает 20, пользователю необходимо удалить старые рабочие записи, прежде чем добавлять новые.
  • Рабочие записи доступны для просмотра и редактирования всем пользователям.

Процедура

Табл. 1 Операции с рабочими записями

Операция

Процедура

Добавление рабочей записи
  1. Нажмите Добавить.
  2. В текстовом поле введите описание выполненной работы и нажмите Сохранить.

Редактирование рабочей записи
  1. Выберите рабочую запись, которую хотите отредактировать.
  2. Нажмите и измените содержимое рабочей записи в текстовом поле.
  3. Нажмите Сохранить.

Удаление рабочей записи
  1. Выберите рабочую запись, которую хотите удалить.
  2. Нажмите .
  3. В появившемся диалоговом окне подтверждения нажмите Да.
