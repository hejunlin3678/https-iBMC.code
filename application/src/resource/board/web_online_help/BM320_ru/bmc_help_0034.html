Виртуальный носитель

Описание функций

Служба VMM (Virtual Media Manager) используется для подключения виртуальных CD-, DVD- и флоппи-дисков к серверу.

На странице Виртуальный носитель можно посмотреть и настроить параметры VMM.

Описание параметров

Табл. 1 Виртуальный носитель

Параметр

Описание

VMM

Статус службы VMM.

Данная функция включена по умолчанию.

Порт

Номер порта для службы VMM.

Значения: от 1 до 65535

Значение по умолчанию: 8208

ПРИМЕЧАНИЕ:

Для обеспечения безопасности браузер отключает некоторые порты, которые не используются для просмотра веб-страниц. Если эти порты используются, виртуальный носитель на интегрированной удаленной консоли HTML5 будет недоступен.

Максимальное количество сеансов

Максимальное количество пользователей, которым одновременно разрешено использовать виртуальный носитель (виртуальный DVD-диск или флоппи-диск) на удаленной виртуальной консоли. Параметр имеет фиксированное значение 1.

Активные сеансы

Количество пользователей, которые используют виртуальный носитель (виртуальный DVD-диск или флоппи-диск) на удаленной виртуальной консоли.
