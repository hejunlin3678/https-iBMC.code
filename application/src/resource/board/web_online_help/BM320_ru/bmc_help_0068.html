Управление Smart Provisioning

Описание функций

Smart Provisioning – это интеллектуальный инструмент, используемый для развертывания серверов.

На странице Управление Smart Provisioning доступны следующие операции:
  • Настройка функции сбора информации о сервере.
  • Настройка функции загрузки ОС с SP.

Описание параметров

Табл. 1 Управление Smart Provisioning

Параметр

Описание

Сбор информации о сервере

Если данная функция включена, внутриполосная система получает доступ к Smarting Provisioning для сбора информации о сервере, когда внутриполосная система запускается в первый раз после включения блоков питания. Затем сервер запускается на основе настроек Приоритетный носитель для загрузки и Приоритет загрузки в разделе Настройки BIOS.

По умолчанию данная функция отключена.

Загрузка ОС с SP

После включения этой функции внутриполосная система напрямую входит в SP после перезапуска сервера.

По умолчанию данная функция отключена.

ПРИМЕЧАНИЕ:

Настройка вступает в силу только при следующем перезапуске. Эта функция автоматически отключается, когда система входит в Smart Provisioning.
