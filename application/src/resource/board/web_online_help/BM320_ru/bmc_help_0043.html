Управление лицензиями

Описание функций

На странице Управление лицензиями после авторизации лицензии пользователь получает доступ к расширенным функциональным возможностям BMC.

Расширенные функции BMC доступны в течение срока действия лицензии. Без лицензии пользователь получает доступ только к стандартному набору функций BMC.

BMC предоставляет следующие расширенные функциональные возможности:
  • Развертывание ОС посредством интерфейса Redfish.
  • Сбор первоначальных данных для интеллектуальной диагностики посредством интерфейса Redfish.

  • Включение механизмов ускорения Kunpeng, включая безопасный ускоритель (SEC, Security Engine), высокопроизводительный RSA-ускоритель (HPRE, High Performance RSA Engine), ускоритель RAID DIF (RDE) и ускоритель ZIP.

  • Включение режим блокировки системы.

Статус лицензии

Лицензия может иметь один из следующих статусов:

  • Обычный: коммерческая лицензия установлена, срок действия лицензии не истек, и все авторизованные функции доступны для использования.
  • Наладка: лицензия на наладку установлена, срок действия лицензии не истек, и все авторизованные функции доступны для использования.
  • Льготный период: коммерческая лицензия или лицензия на наладку установлена, срок действия лицензии истек, наступил льготный период, после истечения которого все авторизованные функции станут недоступными.
  • По умолчанию: коммерческая лицензия или лицензия на наладку установлена, но льготный период лицензии истек.

Код отзыва

Код, сгенерированный после нажатия кнопки Отозвать для установленной лицензии. Этот код можно использовать для подачи заявки на новую лицензию.

Данные о лицензии

Данные о лицензии включают следующую информацию:

  • SN лицензии
  • Тип лицензии
  • Доступны следующие два типа лицензий:
    • Коммерческая: официальная лицензия, которая выдается для клиента на основании договора. Все авторизованные функции имеют одинаковый срок действия. Лицензия может быть Бессрочная или на определенный срок. Когда срок действия лицензии истекает, наступает льготный период, который составляет 60 дней.
    • Пробная: временная лицензия, предназначенная для пробного использования новых функций, диагностики системы или рекламы бренда. Срок действия варьируется в зависимости от реальной ситуации. Когда срок действия лицензии истекает, наступает льготный период, который составляет 60 дней.
  • Срок истечения: дата завершения действия лицензии. Лицензия может быть Бессрочная или на определенный срок.
  • Уровень лицензии
    • Стандартная версия (по умолчанию): лицензия на функции, предоставляемые стандартной версией BMC. Пользователям не нужно покупать данную версию лицензии.
    • Пробная версия: лицензия на функции, предоставляемые расширенной версией BMC. Пользователи могут использовать эти функции только после покупки лицензии.
  • Льготный период: количество дней, в течение которых функции BMC можно использовать после истечения срока действия лицензии. Льготный период имеет фиксированное значение 60 дней.
