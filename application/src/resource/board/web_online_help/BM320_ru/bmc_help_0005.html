Вход в систему

Описание функций

Доступ к WebUI BMC можно получить со страницы входа.

  • BMC поддерживает одновременный вход в WebUI четырех пользователей.
  • По умолчанию время ожидания системы составляет 5 минут. Если пользователь не выполняет в WebUI никаких операций в течение 5 минут, его сеанс завершится автоматически.
  • Если количество последовательных попыток ввода неправильного пароля достигнет заданного значения, система заблокирует аккаунт пользователя. Пользователь может войти в систему только после истечения срока блокировки.
  • В целях обеспечения безопасности рекомендуется сменить первоначальный пароль после первого входа в систему, а затем периодически его менять.
  • Неверное отображение страницы WebUI BMC может возникнуть в том случае, когда не удается получить доступ к ресурсам из-за колебаний напряжения сети. В этом случае обновите браузер и выполните повторный вход в WebUI BMC.

Описание параметров

Табл. 1 Параметры на странице входа

Параметр

Описание

Имя пользователя

Поле для ввода имени пользователя при входе в WebUI BMC.

  • Если в поле Домен выбран Локальный BMC, имя пользователя должно состоять максимум из 16 символов.
  • Если в поле Домен выбрано другое значение, имя пользователя может состоять из 255 символов.

Обратите внимание:

  • При входе в BMC в качестве локального пользователя выберите в поле домена Локальный BMC или Автоподбор.
  • Для пользователя LDAP поддерживаются следующие форматы имени пользователя:
    • Имя пользователя LDAP (в поле домена выберите Автоподбор или введите имя домена)
    • Имя пользователя LDAP@Имя домена (в поле домена выберите Автоподбор или введите имя домена)
  • Для пользователя Kerberos поддерживаются следующие форматы имени пользователя:
    • Имя пользователя Kerberos (в поле домена выберите Автоподбор или введите имя домена)
    • Имя пользователя Kerberos@Имя домена (в поле домена выберите Автоподбор или введите имя домена). Имя домена необходимо вводить заглавными буквами.
  • BMC поддерживает SSO с помощью имени пользователя Kerberos или в формате имя пользователя Kerberos@Имя домена.

Пароль

Поле для ввода пароля при входе в WebUI BMC.

ПРИМЕЧАНИЕ:

Максимальная длина пароля для пользователей LDAP или Kerberos, которые входят в веб-интерфейс BMC, составляет 255 символов.

Вход в качестве локального пользователя

  1. На странице входа в WebUI BMC выберите язык.
  2. Введите имя пользователя и пароль.
  3. Выберите в выпадающем списке Локальный BMC или Автоподбор.
  4. Нажмите Войти.

    Откроется страница Информация, в правом верхнем углу которой будет отображаться имя пользователя, выполнившего вход.

Вход в качестве пользователя LDAP

Перед входом в систему убедитесь, что настройки соответствуют следующим требованиям:

  • В сети имеется контроллер домена, а в контроллере домена созданы домен пользователей и пользователи LDAP.

    Более подробная информации о создании контроллера домена, домена пользователей и пользователей LDAP приводится в документации по контроллеру домена. BMC предоставляет доступ только пользователям LDAP.

  • Протокол LDAP включен в WebUI BMC на странице Пользователи и безопасность > LDAP. домен пользователей и пользователи LDAP настроены.
  1. На странице входа в WebUI BMC выберите язык.
  2. Введите имя и пароль пользователя LDAP.

    • Имя пользователя LDAP может быть представлено в любом из следующих форматов:
      • Имя пользователя LDAP

        После ввода имени пользователя LDAP параметру Домен можно установить значение Автоподбор или ввести определенный домен.

      • Имя пользователя LDAP@имя домена

        После ввода имени пользователя в формате Имя пользователя LDAP@имя домена параметру Домен можно установить значение Автоподбор или ввести определенный домен.

    • Максимальная длина пароля для входа пользователя LDAP в WebUI BMC составляет 255 символов.

  1. Выберите домен пользователей LDAP в выпадающем списке Домен.

    Выпадающий список Домен содержит следующие опции:

    • Локальный BMC: данная опция используется для входа локальных пользователей. BMC автоматически находит аккаунт пользователя в списке локальных пользователей.
    • Настроенный сервер домена: данная опция используется для входа пользователей LDAP. BMC автоматически находит аккаунт пользователя на сервере домена.
    • Автоподбор: при выборе данной опции BMC сначала осуществляет поиск аккаунта пользователя в списке локальных пользователей. Если аккаунт не был найден, BMC выполняет поиск на серверах доменов в той последовательности, в которой серверы доменов расположены в выпадающем списке Домен.

  2. Нажмите Войти.

    Откроется страница Информация, в правом верхнем углу которой будет отображаться имя пользователя, выполнившего вход.

Вход в качестве пользователя Kerberos

Рабочая среда Kerberos:

  • На клиенте установлены ОС Windows 10 (64 бит)1.
  • Сервер Kerberos поддерживает ОС Windows Server 2012 R2 (64 бит) и Windows Server 2016 (64 бит).

Поддерживаются следующие режимы входа:

  • Вход с помощью аккаунта домена Kerberos
  • SSO

Перед тем, как выполнить вход, убедитесь, что выполняются следующие условия:

  • Протокол Kerberos включен в WebUI BMC на странице Пользователи и безопасность > Kerberos. Параметры функции Kerberos и группы пользователей сконфигурированы.
  • Группа пользователей Kerberos и аккаунт пользователя были созданы на сервере Kerberos, и пользователь данного аккаунта был добавлен в созданную группу пользователей Kerberos. Пользователь имеет права на вход в ОС клиента.

Для входа в WebUI BMC с помощью аккаунта домена Kerberos выполните следующие действия:

  1. На странице входа в WebUI BMC выберите язык.
  2. Введите имя и пароль пользователя Kerberos. Более подробная информация приводится в разделе Описание параметров.
  3. В выпадающем списке доменов выберите домен пользователей Kerberos, например ADMIN.COM(KRB), или значение Автоподбор.
  1. Нажмите Войти.

    После успешного входа отобразится Главная страница.

Для входа с помощью SSO выполните следующие действия:

  1. Используйте имя и пароль пользователя Kerberos, сконфигурированные на сервере Kerberos для входа в ОС клиента.
  2. Откройте браузер, введите в адресной строке FQDN BMC, например https://Имя хоста.Имя домена, и нажмите Enter.

    Откроется страница входа BMC.

  3. Нажмите SSO.

    После успешного входа отобразится Главная страница.
