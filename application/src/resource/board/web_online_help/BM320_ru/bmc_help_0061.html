Описание параметров

Табл. 1 Другое

Категория

Описание

BBU
Информация о платах BBU (только для серверов с поддержкой BBU):

  • Максимальное и фактическое количество BBU

  • Название, идентификатор, версия встроенного ПО, остаточный уровень заряда аккумуляторной батареи, рабочий статус, модель аккумуляторной батареи, серийный номер и производитель каждой платы BBU

Плата ЦПУ
Информация о плате ЦПУ:
  • Максимальное и фактическое количество плат ЦПУ
  • Название, производитель, номер слота, тип, версия PCB, версия CPLD, UID компонента, идентификатор платы и мощность каждой платы ЦПУ

Соединительные платы накопителей

Информация о соединительных платах накопителей:

  • Максимальное количество расширяемых соединительных плат накопителей и количество установленных соединительных плат накопителей.
  • Название, расположение, производитель, номер, тип, версия PCB, версия CPLD, идентификатор платы, UID компонента, BOM-код и серийный номер каждой соединительной платы

Expander

Предоставляет информацию о плате Expander (только для сервера, поддерживающего платы Expander):

  • Максимальное и фактическое количество плат Expander
  • Имя, ID платы, UID компонента, Версия CPLD, №, Описание, местоположение, производитель, версия PCB, BOM-код и Серийный номер Expander

Соединительная плата вентиляторов

Информация о cоединительная плата вентиляторов:

  • Максимальное и фактическое количество cоединительная плата вентиляторов
  • Имя, mестоположение, производитель, UID компонента, тип, версия PCB и ID платы cоединительная плата вентиляторов

Адаптер M.2

Информация об адаптере M.2, включая название, описание, слот, версию PCB, ID платы, BOM-код и серийный номер(только для серверов с поддержкой Адаптер M.2).

Платы OCP
Информация о платах OCP (только для серверов с поддержкой Платы OCP):
  • Максимальное и фактическое количество плат OCP
  • Описание, расположение, производитель, номер слота, идентификатор вендора, идентификатор устройства, идентификатор субвендора, идентификатор подустройства и подключенный ресурс каждой карты OCP

Платы PCIe

Информация о платах PCIe:

  • Максимальное и фактическое количество плат PCIe
  • Описание, расположение, производитель, номер слота, идентификатор вендора, идентификатор устройства, BOM-код, идентификатор субвендора, идентификатор подустройства и подключенный ресурс каждой платы PCIe

Райзер-карты

Информация о райзер-картах:

  • Максимальное и фактическое количество райзер-карт
  • Название, производитель, слот, тип, версия PCB, UID компонента, идентификатор платы, BOM-код и серийный номер каждой райзер-карты

Модуль системы безопасности

Информация о модулях системы безопасности:

  • Максимальное и фактическое количество модулей системы безопасности
  • Тип и версия протокола, производитель, производственная версия, a также состояние самотестирования каждого модуля системы безопасности
