Видео и скриншоты

Описание функций

В разделе Проигрывание видео доступны следующие операции:

  • Включение или отключение функции записи видео.

    При включении данной функции BMC автоматически записывает видео ошибки ЦПУ, видео выключения и видео перезагрузки.

  • Воспроизведение видеофайла сервера, хранящегося на локальном ПК.
  • Воспроизведение видео, автоматически записанного на сервере.
  • Создание скриншотов во время воспроизведения видео.
    • Видеофайл должен быть в формате *.rep.
    • Скриншоты должны быть в формате *.jpg.
    • Обратите внимание, что после включения функции видеозаписи может быть записана конфиденциальная служебная информация во время видеозаписи.

В разделе Скриншот доступны следующие операции:

  • Включение или отключение функции скриншота последнего сбоя.

    Функция скриншота последнего сбоя позволяет автоматически создавать скриншот экрана перед выключением или перезагрузкой сервера.

  • Создание скриншотов рабочего стола сервера в режиме реального времени.

    Функция скриншота последнего сбоя включена по умолчанию. Обратите внимание, что после включения данной функции может быть записана конфиденциальная служебная информация в процессе создания скриншота.

Загрузка видеофайла

Нажмите в поле Видео ошибки ЦПУ, Видео выключения или Видео перезагрузки для загрузки видеофайла в каталог по умолчанию локального ПК.

ПРИМЕЧАНИЕ:

Если размер видеофайла превышает 10 МБ, рекомендуется скачать и сохранить его на локальном ПК. BMC удалит видеофайл размером более 10 МБ во время сбора информации одним щелчком мыши.

Воспроизведение локального видеофайла
  1. Выберите одного из следующих проигрывателей:
    • Запустите проигрыватель HTML5 в поле Загрузить проигрыватель.
    • Запустите проигрыватель Java, загруженный из поля Загрузить проигрыватель.
  2. На видеопроигрывателе нажмите и выберите видеофайл на локальном ПК.
  3. Нажмите Открыть.

    Начнется воспроизведение видеофайла.

  4. (Опционально) Настройте состояние воспроизведения видео по мере необходимости.
    • Для быстрой перемотки видео с 1-, 2- или 4-кратным ускорением нажмите .
    • Для обратной перемотки видео с 1-, 0,5- или 0,25-кратным ускорением нажмите .
    • Для управления процессом воспроизведения видео перемещайте влево или вправо.
    • Для повторного воспроизведения видео нажмите .
    • Для отображения окна видеопроигрывателя в полноэкранном режиме нажмите .

Воспроизведение видеофайла онлайн
  1. Выберите одного из следующих проигрывателей:
    • Запустите проигрыватель HTML5 в поле Видео ошибки ЦПУ, Видео выключения или Видео перезагрузки.
    • Запустите проигрыватель Java, загруженный из поля Видео ошибки ЦПУ, Видео выключения или Видео перезагрузки.
  2. (Опционально) Настройте состояние воспроизведения видео по мере необходимости.
    • Для быстрой перемотки видео с 1-, 2- или 4-кратным ускорением нажмите .
    • Для обратной перемотки видео с 1-, 0,5- или 0,25-кратным ускорением нажмите .
    • Для управления процессом воспроизведения видео перемещайте влево или вправо.
    • Для отображения окна видеопроигрывателя в полноэкранном режиме нажмите .

Создание скриншота экрана во время воспроизведения видео

Нажмите во время воспроизведения видео.

Сохраните скриншот на клиенте. Скриншоты сохраняются в формате *.jpg.

Просмотр последнего скриншота

Нажмите на миниатюру в разделе Скриншот для увеличения скриншота.

Система поддерживает отображение максимум трех миниатюр с соответствующими отметками времени.

Создание скриншота
  1. Нажмите в поле Создать скриншот.

    Откроется диалоговое окно подтверждения.

  2. Нажмите Подтвердить.

    На экране отобразится скриншот рабочего стола сервера. В верхнем левом углу скриншота будет указано время его создания.

    ПРИМЕЧАНИЕ:

    При создании нескольких скриншотов на экране отобразится только последний скриншот и время его создания.

Удаление скриншота
  1. Нажмите в поле Создать скриншот.

    Откроется диалоговое окно подтверждения.

  2. Нажмите Подтвердить.
