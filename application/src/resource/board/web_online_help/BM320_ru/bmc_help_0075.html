Редактирование конфигураций

Только администратор может редактировать конфигурации, когда TSB включен и система не заблокирована.

  1. Нажмите Конфигурация.
  2. Редактируйте конфигурацию в появившемся окне Конфигурация.

    • Нажмите кнопку справа от Измерение. Включение или отключение Измерение.
      • Значение по умолчанию включено.
      • Если функция измерения отключена, злонамеренное вмешательство в микропрограмму не может быть обнаружено сразу.
      • После включения функции измерения необходимо обновить базовое значение микропрограммы при обновлении микропрограммы.
    • Нажмите кнопку справа от Контроль. Включение или отключение Контроль.
      • Значение по умолчанию отключено.
      • Если функция контроля отключена, микропрограмма не может быть немедленно остановлена при злонамеренном вмешательстве в нее, что приводит к рискам безопасности системы.
      • Функция контроля можно редактировать только в том случае, если функция измерения включена.
      • После включения функции контроля, если микропрограмма не проходит измерение, запуск микропрограммы будет заблокирован.
    • Введите адрес в текстовом поле Адрес доверенного сервера. Редактирование Адрес доверенного сервера.

      Формат адреса: IPv4-адрес, IPv6-адрес или доменное имя.

      Имя домена должно соответствовать следующим требованиям:
      • Cтрока длиной до 255 символов.
      • Имя домена может состоять только из цифр, букв, дефисов (-) и точек (.).
      • Имя домена не может начинаться с дефиса (-) или точки (.), либо заканчиваться дефисом (-).
      • Имя домена может состоять из не более 63 символов между любыми двумя точками.

  3. Нажмите OK.

    Появится диалоговое окно подтверждения.

  4. Введите пароль для входа в систему и нажмите OK.

Обновление базового значения

  • Только администратор может обновить базовое значение, когда TSB включен и режим блокировки системы отключен.
  • Обновляйте базовое значение только при обновлении микропрограммы. В противном случае не рекомендуется изменять базовое значение.

  1. Нажмите кнопку Обновить.

    На экране появится диалоговое окно Обновить.
    • Если микропрограммное обеспечение BIOS не обновляется с помощью BMC, необходимо сначала включить ОС после завершения обновления микропрограммного обеспечения BIOS, а затем перезагрузить BMC, чтобы отображаемое значение измерения отражало последние измерения микропрограммного обеспечения. Затем обновите базовое значение.
    • Чтобы обновить базовое значение других микропрограмм, перезагрузите BMC после обновления микропрограммы и убедитесь, что отображаемое фактическое значение измерения является последней информацией об измерении микропрограммы. Затем обновите базовое значение.

  2. (Опционально) Измените Базовое значение в текстовом поле.

    Базовое значение состоит из:

    • От 1 до 64 символа.
    • Цифры (0-9) и буквы (a-f, A-F).

  3. Нажмите кнопку Сохранить.

    На экране появится диалоговое окно с просьбой подтвердить ваши действия.

  4. Введите пароль для входа текущего пользователя и нажмите кнопку "Да".

    • Базовое значение микропрограммного обеспечения BIOS обновлено. Снова перезагрузите BMC, чтобы базовое значение стало эффективным, чтобы фактическое значение измерения соответствовало базовому значению.
    • После обновления базового значения других микропрограмм, перезагрузите ОС, чтобы базовое значение вступило в силу, и фактическое значение измерения соответствовало базовому значению.
