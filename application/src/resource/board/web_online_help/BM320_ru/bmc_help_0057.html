Информация о процессоре

Информация о каждом процессоре, установленном в сервере:

  • Количество процессоров в полной конфигурации и количество установленных процессоров
  • Имя, производитель, модель, идентификатор процессора, тактовая частота, BOM-код и серийный номер каждого процессора
  • Количество ядер и потоков
  • Емкость кэша L1, L2 и L3
  • Статус процессора
  • Серийный номер
  • Другие параметры
