BIOS構成

機能説明

[[BIOS構成]]画面で、サーバー上のOSの最初のブートデバイスを設定します。

パラメータの説明

表 1 ブートデバイス

カテゴリ

説明

優先起動媒体
  • [HDD]: このオプションを選択すると、ハードディスクからOSを起動できます。
  • [CD/DVD]: このオプションを選択すると、CD-ROMまたはDVD-ROMドライブからOSを起動できます。
  • [FDD/リムーバブルデバイス]: このオプションを選択すると、フロッピーディスクドライブ（floppy disk drive、FDD）または取り外し可能なデバイスからOSを起動できます。
  • [PXE]: このオプションを選択すると、PXE（Preboot Execution Environment）からOSを起動できます。
  • [BIOS設定]: このオプションを選択すると、サーバーの起動時にBIOS構成メニューを表示できます。
  • [未構成] このオプションを選択すると、OSはBIOSで指定された最優先の起動媒体から起動します
  • [1回のみ有効]: 指定された優先起動媒体は、サーバーの次回再起動時にのみ使用されます。サーバーの再起動後、[優先起動媒体]の値は自動的に[未構成]に復元されます。
  • [常に有効]: 優先起動媒体の設定は永続的に有効になります。

デフォルト値は[未構成] と[1回のみ有効]です。

起動順位

[優先起動媒体]の値が[未構成]に設定されている場合、OSは[起動順位]の設定に従って起動します。

をクリックしてオプションを上に移動したり、をクリックしてオプションを下に移動したりすることができます。

NOTE:
  • BIOSでの起動優先度設定は、直ちに有効になります。OSを再起動すると、BMCの起動優先度設定はBIOSの設定と同期されます。
  • BMCでの起動優先度設定は、OSの再起動後に有効になります。OSを再起動すると、BIOSの起動優先度設定はBMCの設定と同期されます。

ブートデバイスの設定

  1. [ブートデバイス]タブで、サーバーOSの最初のブートデバイスを設定します。表 1にパラメータの説明を示します。
  2. [保存]をクリックします。
