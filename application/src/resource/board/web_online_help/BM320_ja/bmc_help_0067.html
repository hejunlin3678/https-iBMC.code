USB管理

機能説明

[USB管理]画面では、BMCダイレクトコネクト管理ポートの照会および設定を実行できます。

USB管理機能は、サーバー上のBMCダイレクトコネクト管理ポートを通して利用する必要があります。

パラメータの説明

表 1 USB管理

カテゴリ

説明

USB管理

この機能が有効になっている場合、サーバの取り付け金具にあるUSBポートを介してBMCにアクセスすることができます。

  • ローカルPCの場合：ブラウザを使用するか、ローカルPCからSSHでBMC CLIにアクセスすることで、BMCのWebUI（IPアドレス169.254.1.5）にアクセスできます。
  • モバイル端末：モバイルアプリケーションSmartServerを使用することで、モバイル端末からBMCにアクセスできます。
NOTE:
  • このパラメータは、デフォルトでは有効になっています。です。
  • Windows 10を搭載したPC、Androidを搭載したスマートフォンのみがこの機能をサポートしています。

USB機器状態

USB機器の状態を示します。

  • 接続済み：USBポート経由で接続されている機器があります。
  • 未接続：USBポート経由で接続されている機器がありません。
