VNC

機能説明

VNCを使用することで、ローカルPC（クライアント）のキーボード、マウス、モニタを使用してサーバーにアクセスできます。これにより、サーバーをリアルタイムで管理できます。

[VNC]画面で、VNC設定を照会・構成できます。

パラメータの説明

表 1 VNC

パラメータ

説明

VNC

VNCサービスの状態。

デフォルトで有効になっています。

ポート

VNCサービスのポート番号。

値は以下のとおりです。1～65535

デフォルト値: 5900

タイムアウト期間（分）

VNCインターフェースでの最後の操作後の最大アイドル期間（分）。何の操作も実行しないまま、タイムアウト制限時間が経過すると、ユーザーはVNCインターフェースからログアウトされます。

値の範囲: 0～480

値0は、時間が無制限であることを意味します。

デフォルト値: 60

キーボードレイアウト

VNCによって制御されるOSのキーボードレイアウト。

値は以下のとおりです。

  • 日本語
  • 英語（US）
  • ドイツ語

デフォルト値: 日本語

VNCパスワード

VNCインターフェースにログイン用のパスワード。

値は以下のとおりです。

  • パスワード複雑度チェック機能が無効になっている場合、VNCパスワードには数字、英字、特殊文字を含む1～8桁の文字列である必要があります。
  • パスワード複雑度チェック機能が有効になっている場合、VNCパスワードは次の要件を満たす必要があります:
    • 8文字です。
    • 複雑さの要件:
      • スペースと`~!@#$%^&*()-_=+\|[{}];:'",<.>/?のうち、少なくとも1つを含める必要があります。

        `~!@#$%^&*()-_=+\|[{}];:'",<.>/?

      • 大文字（A～Z）、小文字（a～z）、数字（0～9）のうち、2種類を含める必要があります。

VNCパスワードの確認

確認のためにコミュニティ名を再入力します。

パスワードの有効期限（日数）

VNCパスワードの有効期間。

ログインルール

VNCユーザーに適用されるログインルール。

ローカルユーザーのために設定されたログインルールは、VNCユーザーにも適用されます。

SSL暗号化

SSL暗号化を有効または無効にする機能。

デフォルトで有効になっています。

セキュリティ上の理由から、この機能を有効にすることをお勧めします。SSL暗号化が無効になっている場合、VNCクライアントはRFB（Remote Frame Buffer）プロセスを開始します。

NOTE:

SSL暗号化を有効にした場合、SSL暗号化を有効にしたVNCクライアントのみがサーバーOSに接続できます。VNCクライアントがSSL暗号化オプションを提供していない場合は、SSLトンネリングアプリケーションを使用してSSL暗号化を実装してください。

最大セッション数

VNCインターフースへ同時にアクセスできるユーザーの最大数。

このパラメータは5に指定されています。

アクティブセッション数

現在VNCインターフェースに同時にアクセスしているユーザーの数。
