診断

機能説明

[診断]画面でPCIeポートの有効化または無効化を行うことができます。PCIeインターフェースを有効にすると、インバンドおよびアウトバンドのシステム通信とブラックボックスデータ収集に使用できます。MCTPがサポートされている場合、PCIeインターフェースはMCTP通信にも使用できます。

[診断]画面でブラックボックスの有効化または無効化を行うことができます。ブラックボックスを有効にした場合、ブラックボックスに保存されているデータをローカルホストにダウンロードすることができます。

ブラックボックスは、メモリーと障害監視ソフトウェアで構成されています。

  • メモリーは、サーバー上のハードディスクに依存しない障害情報を記録する内蔵ストレージチップです。

    メモリーは最大4MBのストレージ容量を提供し、OSがクラッシュしたときにカーネル情報を保存します。

  • 障害監視ソフトウェアは、OSがクラッシュしたときのカーネル情報を記録します。

    ブラックボックス機能を使用する前に、障害監視ソフトウェア（例:BMA）がサーバーにインストールされていることを確認してください。BMAのインストールと使用方法については、BMA ユーザー Guideを参照してください。

  • ブラックボックス機能が有効になった後、ブラックボックスドライバーがサーバーにインストールされていない場合、OS上で未知のデバイスが検出される可能性があります。

診断画面で、シリアルポート経由で送信されたデータを記録する機能を有効または無効にし、最新のデータをダウンロードできます。

操作

手順

PCIeポートの有効化／無効化
  1. PCIeポート機能を有効にするには、PCIeポートをに設定します。PCIeポート機能を無効にするには、PCIeポートをに設定します。またはをクリックして設定を変更します。
  2. サーバーを再起動します。
NOTE:
  • PCIeポートはデフォルトで有効になっています。
  • PCIeポートの設定を変更した場合、設定を有効にするにはサーバーを再起動する必要があります。

ブラックボックスデータのダウンロード

ブラックボックス機能が有効になっていることを確認します。

ブラックボックスエリアで、をクリックします。

ブラックボックスのデータはローカルPCのデフォルトディレクトリに自動的にダウンロードされます。

NOTE:
  • [ブラックボックス]はデフォルトで有効になっています。
  • BMCはサーバーからブラックボックスデータファイルをローカルPCにダウンロードしますが、このファイルを解析しません。ブラックボックスデータファイルの解析方法については、BMA ユーザー Guideを参照してください。
  • 表示されたファイル保存情報は、お使いのブラウザによって異なります。
表 2 シリアルポートデータの操作

操作

手順

システムシリアルポートデータの記録機能の有効化/無効化

この機能を有効にするには、シリアルポートデータに設定します。この機能を無効にするには、シリアルポートデータをに設定します。またはをクリックして設定を変更します。

NOTE:

シリアルポートデータ機能はデフォルトで有効になっています。

システムのシリアルポートデータのダウンロード

シリアルポートデータに設定されている場合にのみ、シリアルポートデータをダウンロードします。

シリアルポートデータエリアにあるをクリックします。

システムのシリアルポートデータは、ローカルPCのデフォルトディレクトリに自動的にダウンロードされます。

NOTE:
  • システムシリアルポートの最新のデータがダウンロードされます。
  • 表示されたファイル保存情報は、お使いのブラウザによって異なります。
