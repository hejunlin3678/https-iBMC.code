カテゴリ

説明
  • ネットワーク情報は、BMA 2.0が起動した後にのみネットワークアダプター画面に表示できます。
  • BMA 2.0がサーバーにインストールされていない場合は、最新のBMAドキュメントとソフトウェアパッケージを入手し、BMA 2.0をインストールしてください。

イーサネットNIC

LOMおよびPCIe NICに関する情報であり、NICの名称、製造メーカー、型番、チップモデル、チップベンダー、PCBバージョン、ボードID、コンポーネントUID、接続先（CPU、PCHまたはPCIeスイッチ）が含まれます、バス情報、PCIeスロット情報を含みます。

  • イーサネットNIC名称をクリックすると、ポート番号、ポート状態、ポートタイプ、メディアタイプ、速度、オートネゴシエーションおよび全二重の状態などのポートの詳細情報が表示されます。
    NOTE:

    [ポートプロパティ]エリア内のポート状態は以下のいずれかになります。

    • --：サーバーにBMAをインストールしておらず、物理的な接続状態を取得できません。
    • 接続：サーバーにBMAをインストールしておらず、物理リンクが接続されています。
    • 切断：サーバーにBMAをインストールしておらず、物理リンクが接続されていません。
    • NoLink：サーバーにBMAをインストールしており、ポートが接続されていないが、ポート状態が[Up]になっています。
    • LinkUp：サーバーにBMAをインストールしており、ポートが接続され、ポート状態が[Up]になっています。
    • LinkDown：サーバーにBMAをインストールしており、ポート状態が[Down]になっています。
  • ポートプロパティの下にあるをクリックすると、ネットワークプロパティが表示されます。ネットワークパロパティには、各ネットワークポートの名称、ファームウェアバージョン、ドライバーの名称とバージョン、バス情報、MACアドレス、永続的MACアドレス、IPv4情報（アドレス/サブネットマスク/ゲートウェイ）、IPv6情報（アドレス/プレフィクス長/ゲートウェイ）およびVLAN情報（VLAN ID、VLANの有効設定、VLAN優先度の有効設定）が含まれます。
  • NIC接続ビューがサポートされます。ポートプロパティの下にあるをクリックすると、接続ビューが表示されます。接続ビューには、スイッチ名、スイッチ接続ID、スイッチに接続されているポートのID、およびスイッチポートVLAN IDに関する情報が表示されます。
  • ポートプロパティの下にあるをクリックすると、NICのDCB情報とイーサネット統計が表示されます。
  • ネットワークポートに光モジュールを接続している場合、ポートプロパティの下にあるをクリックすると、光モジュール情報およびとアラームしきい値情報が表示されます。光モジュール情報には、メーカー、シリアル番号、部品名、デバイスタイプ、コネクタタイプ、受信パケットロス、送信パケットエラー、波長、デバイス識別子、現在の温度、現在の入力バイアス電流、現在の出力電力、現在の入力電力が含まれます。
  • ネットワークポートに電力ケーブルを接続している場合、ポートプロパティの下にあるをクリックすると、電力ケーブル情報が表示されます。電力ケーブル情報には、メーカー、シリアル番号、部品名、デバイスタイプ、コネクタタイプが含まれます。
NOTE:

NICのファームウェアバージョンがネットワークポートをサポートしていない場合、ポートのネットワークプロパティは空です。たとえば、NICにPort1とPort2の2つのポートがあります。NICのファームウェアがPort2に対応しない場合、Port2のネットワークプロパティは空です。

FCアダプター
アダプターの名称、製造メーカー、型番、チップモデル、チップメーカーな、部品番号、シリアル番号どのFCアダプター情報を示します。
  • FCカードの名称をクリックすると、ポート番号、FC ID、ポートタイプ、状態などのポートの詳細情報が表示されます。
  • ポートプロパティの下にあるをクリックすると、速度、WWPN（World Wide Port Name）、WWNN（World Wide Node Name）、ファームウェアバージョン、およびドライバーの名称とバージョンが表示されます。
  • ネットワークポートに光モジュールを接続している場合、ポートプロパティの下にあるをクリックすると、光モジュール情報が表示されます。光モジュール情報には、メーカー、シリアル番号、製造日、部品名、モジュールタイプ、送信モード、トランシーバーの種類、速度、デバイスの種類、コネクタタイプ、伝送距離、受信パケットロス、送信パケットエラー、波長、デバイス識別子、現在の温度(℃)、現在の電圧(V)、現在の入力バイアス電流(mA)、現在の出力電力(mW)、現在の受信電力(mW)、メジャーアラーム閾値（上限|下限）が含まれます。
  • Hi1822プロセッサを使用し、MCTPに対応するFCカードでは、以下の情報を表示することができます。
    • 動作速度
    • 動作モード
    • 光モジュールの状態
    • ピアデバイスの信頼性
    • ローカルデバイスの信頼性
    • 送信速度
    • 受信速度
    • 速度のオートネゴシエーション段階

Team

トランクポートの名称、状態、動作モード、IPv4情報（アドレス/サブネットマスク/ゲートウェイ）、IPv6情報（アドレス/プリフィックス長さ/ゲートウェイ）、MACアドレス、VLAN情報（VLAN ID、VLANの有効設定、VLAN優先度の有効設定）を示します。

ポート名称をクリックすると、NIC名称、Eth名称、Syslogポート、MACアドレス、状態などのポートの詳細情報が表示されます。

Bridge

ブリッジポートの名称、状態、IPv4情報（アドレス/サブネットマスク/ゲートウェイ）、IPv6情報（アドレス/プリフィックス長さ/ゲートウェイ）、MACアドレス、VLAN情報（VLANの有効設定、VLAN優先度の有効設定）を示します。

  • ブリッジネットワークポートの名称をクリックすると、ポートの名称、番号、状態、タイプとメディアタイプを含むポートの詳細情報が表示されます。
  • ポートプロパティの下にあるをクリックすると、メンバーポートの詳細情報が表示されます。

OCP NICの取り外し

  1. [システム] > [システム情報]の順に選択し、[ネットワークアダプター]をクリックします。
  2. 左側にあるイーサネットNICリストからOCPカードを選択します。
  3. NICデバイス画面中のをクリックします。

    下記の情報が表示されます。

    通常のホットリムーブを実行しますか？
通常のホットリムーブを実行すると、NICの機能は使用できなくなります。この操作を実行する際には慎重に行ってください。
