パラメータの説明

表 1 その他

パラメータ

説明

BBU
バッテリバックアップユニット（BBU）情報を表示します（BBUをサポートしているサーバーのみ）。

  • BBUモジュールの最大搭載数と現在搭載されている個数。

  • BBUモジュールの名前、ID、ファームウェアバージョン、動作状態、残存電力、バッテリーモデル、シリアル番号及び製造メーカー。

CPUボード
CPUボード情報を提供します。
  • 現在のCPUボードの最大搭載数と現在の搭載数。
  • CPUボードの名前、製造メーカー、スロット番号、種類、PCBバージョン、CPLDバージョン、ボードID、コンポーネントUID、および電源。

HDDバックプレーン

ドライブバックプレーン情報を提供します。

  • ドライブバックプレーンの拡張可能な最大搭載数と現在の搭載数
  • 各ドライブバックプレーンの名前、位置、メーカー、番号、種類、PCBバージョン、CPLDバージョン、ボードID、コンポーネントUID、部品番号、シリアル番号

Expander
エキスパンダーボード情報を提供します（エキスパンダーボードをサポートしているサーバーのみ）。
  • エキスパンダーボードの最大搭載数と現在の搭載数。
  • エキスパンダーボードの名前、ボードID、コンポーネントUID、CPLDバージョン、番号、詳細、ロケーション、製造メーカー、PCBバージョン、部品番号、およびシリアル番号。

ファンバックプレーン

ファンバックプレーン情報を提供します。

  • ファンバックプレーンの最大搭載数と現在の搭載数。
  • ファンバックプレーンの名前、ロケーション、メーカー、タイプ、PCBバージョン、およびボードID、およびコンポーネントUID。

M.2アダプタ

名前、説明、スロット、PCBバージョン、ボードID、部品番号、およびシリアル番号のM.2アダプタ情報を提供します(M.2アダプタをサポートしているサーバーのみ）。

OCPカード
OCPカード情報を提供します（OCPカードをサポートしているサーバーのみ）。
  • OCPカードの最大搭載数と現在の搭載数。
  • OCPカードの詳細、ロケーション、製造メーカー、スロット番号、ベンダーID、デバイスID、サブベンダーID、サブデバイスID、および接続先。

PCIeカード

PCIeカード情報を提供します。

  • PCIeカードの最大搭載数と現在の搭載数。
  • PCIeカードの詳細、ロケーション、製造メーカー、スロット番号、ベンダーID、デバイスID、部品番号、サブベンダーID、サブデバイスID、および接続先。

Riserカード

Riserカード情報を提供します。

  • Riserカードの最大搭載数と現在の搭載数
  • 各Riserカードの名前、メーカー、スロット、種類、PCBバージョン、ボードID、コンポーネントUID、部品番号、シリアル番号

セキュリティモジュール

セキュリティモジュール情報を提供します。

  • セキュリティモジュールの最大搭載数と現在の搭載数。
  • セキュリティモジュールのプロトコルタイプ、プロトコルバージョン、製造メーカー、製造メーカーバージョン、およびセルフチェック状態。
