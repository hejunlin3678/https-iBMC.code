すべてのSAS/SATAハードディスクログを含むフォルダ。サブフォルダは、ハードディスクに基づいて名前を付けます。例：Disk0。

  • SATAハードディスクログ。例：SATA_Log、SMARTAttribute、SeagateFARMLog、SeagateFARMLog.bin。
    NOTE:

    SeagateFARMLog.binファイルはFARMログの生データです。BMC V3.03.00.01以降のバージョンではこのログファイルの収集をサポートしていません。

phy

すべてのRAIDコントローラカードとそのExpanderのPHYエラーコードログを含むフォルダ。サブフォルダは、RAIDコントローラカードに基づいて名前を付けます。例：RAID_Card1。

  • RAIDコントローラカードのPHYエラーコードログ

    （例：RAID_Card1_PHY_Error_Count.csv）

  • Expander PHYエラーコードログ

    （例：RAID_Card1_Expander1_PHY_Error_Count.csv）
