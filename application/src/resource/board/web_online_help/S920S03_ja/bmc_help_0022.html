作業記録

機能概要

[作業記録]画面で、作業記録を追加および表示できます。

  • 作業記録は255文字以下に限られ、最大20件まで登録できます。20件に達した場合、新しい作業記録を登録する前に古い作業記録を削除する必要があります。
  • 作業記録はすべてのユーザーに表示され、またすべてのユーザーにより編集可能です。

操作

手順

作業記録の追加
  1. レコード追加をクリックします。
  2. 表示されたテキストボックスに作業記録を追加し、保存をクリックします。

作業記録の修正
  1. 変更するワークレコードにカーソルを移動します。
  2. をクリックしてテキストボックスにある作業記録の内容を変更します。
  3. 保存をクリックします。

作業記録の削除
  1. 削除する作業記録にカーソルを移動します。
  2. をクリックします。
  3. 表示された確認ダイアログボックスで、OKをクリックします。
