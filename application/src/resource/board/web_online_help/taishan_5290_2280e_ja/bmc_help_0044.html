機能概要

機能説明

仮想コンソール画面で、HTML5統合リモートコンソールまたはJava統合リモートコンソールを使用してサーバーのオペレーティングシステムにアクセスできます。

説明

HTML5統合リモートコンソール

HTML5統合リモートコンソールは以下の2種類のモードをサポートします。

  • 専有モードでは、1人のローカルユーザーまたはVNCユーザーのみがBMC経由でサーバーのオペレーティングシステムにアクセスして操作できます。
  • 共有モードでは、2人のローカルユーザーまたは5人のVNCユーザーがBMC経由でサーバーのオペレーティングシステムに同時にアクセスして操作できます。各ユーザーは、他のユーザーが実行した操作を表示できます。

HTML5統合リモートコンソールは以下の機能をサポートします。

  • フローティングボタン、全画面表示ボタン、マウス制御ボタン、画像解像度カーソルボタンによって、画面の表示を容易に設定します。
  • キーボード組み合せキーボタンとキーボードレイアウトボタンによって、入力デバイスを設定します。
  • 電源制御ボタンを使用してサーバーOSを制御し、ビデオ録画ボタンを使用してOS上で実行された操作を記録します。
  • DVDおよびFDDボタンを使用して、イメージファイルまたはローカルフォルダをマウントします。

Java統合リモートコンソール

Java統合リモートコンソールは以下の2種類のモードをサポートします

  • 専有モードでは、1人のローカルユーザーまたはVNCユーザーのみがBMC経由でサーバーのオペレーティングシステムにアクセスして操作できます。
  • 共有モードでは、2人のローカルユーザーまたは5人のVNCユーザーがBMC経由でサーバーのオペレーティングシステムに同時にアクセスして操作できます。各ユーザーは、他のユーザーが実行した操作を表示できます。

Java統合リモートコンソールは以下の機能をサポートします。

  • フローティングボタン、全画面表示ボタン、マウス制御ボタン、画像解像度カーソルボタンによって、画面の表示を容易に設定します。
  • キーボード組み合せキーボタン、キーボードインジケーター、キーボードレイアウトボタンによって、入力デバイスの照会と設定をサポートします。
  • 電源制御ボタンを使用してサーバーOSを制御し、ビデオ録画ボタンを使用してOS上で実行された操作を記録します。
  • 物理DVDドライブ、FDD、イメージファイル、またはローカルフォルダをマウントします。
  • イメージ作成ボタンを使用して、DVDまたはソフトウェアのイメージファイルを作成します。

リモートバーチャルコンソールへのログイン

リモートバーチャルコンソールにOSやBIOSのパスワードを入力するときに、
  • オペレーティングシステムのキーボード設定と実際使用中のキーボードが一致していない場合、実際のキーボードを使用してください。
  • オペレーティングシステムのキーボード設定と実際使用中のキーボードが一致している場合、オペレーティングシステムのキーボードを使用してください。
[Webサイトのセキュリティアラート]が表示される場合、必要に応じてこのメッセージを無視するか、次の操作を実行してこのアラートを対処します。
  • BMCに信頼証明書とルート証明書をインポートします。
  • 信頼証明書がない場合は、BMCを[Javaコントロールパネル]の[例外サイトリスト]に追加します。ただし、この操作ではセキュリティリスクが発生します。
  • BMC WebUIのホームページで、ドロップダウンリストからJava統合リモートコンソールまたはHTML5統合リモートコンソールを選択します。

    共有モードでは、2人のユーザーがリモートコンソールを同時に使用してサーバーにアクセスして操作を行うことができます。各ユーザーは、他のユーザーが実行した操作を表示できます。

    専用モードでは、1人のユーザーのみがリモートコンソールを使用してサーバーにアクセスして操作を行うことができます。このモードを選択すると、手動スクリーンショット機能は使用できません。

  • ブラウザを開き、アドレスバーに次のアドレスを入力してログインします。
    • 方法一：
      • HTML5統合リモートコンソールへのログイン：
        • “https://IP address/remoteconsole?openWay=html5”または“https://IP address/remoteconsole?openway=html5”
        • “https://IP address/remote_access.asp?authParam=key&lp=lang&openWay=html5”または“https://IP address/remote_access.asp?authParam=key&lp=lang&openway=html5”
      • Java統合リモートコンソールへのログイン：
        • “https://IP address/remoteconsole”または“https://IP address/remoteconsole?openWay=jre”または“https://IP address/remoteconsole?openway=jre”
        • “https://IP address/remote_access.asp?authParam=key&lp=lang&openWay=jre”または“https://IP address/remote_access.asp?authParam=key&lp=lang&openway=jre”
      • 「key」はRedfishインターフェースによって設置できます。keyで直接KVMにコネクタできます。
      • 「lp」はコンソールの使用されている言語を示します。
      • 「openWay」パラメーターは、「openway」および「openWay」スタイルのみをサポートします。残りの記述方法を使用すると、Javaコンソールに切り替わります。
    • 方法二：“https://IP address/kvmvmm.asp”
    • 方法三：“https://IP address/login.html?redirect_type=1”

      「IP address」はBMC管理ポートのIPアドレスを示します。
