Paramètres du BIOS

Présentation de la fonction

Sur la page BIOS Settings, vous pouvez définir le premier périphérique de démarrage pour le système d'exploitation sur le serveur.

Description du paramètre

Tableau 1 Paramètres du BIOS

Paramètre

Description

Support de démarrage
  • Disque dur : Sélectionnez cette option pour démarrer le système d'exploitation à partir du disque dur.
  • CD/DVD : Sélectionnez cette option pour démarrer le système d'exploitation à partir du lecteur de CD-ROM ou de DVD-ROM.
  • Lecteur disquette/Périphérique amovible : Sélectionnez cette option pour démarrer le système d'exploitation à partir d'un lecteur de disquettes virtuel (FDD) ou d'un périphérique amovible.
  • PXE : Sélectionnez cette option pour démarrer le système d'exploitation à partir de l'environnement PXE (Preboot Execution Environment).
  • Configuration du BIOS : Sélectionnez cette option pour afficher le menu de configuration du BIOS au démarrage du serveur.
  • Aucun remplacement : Sélectionnez cette option pour démarrer le système d'exploitation à partir du premier périphérique de démarrage spécifié par défaut dans le BIOS.
  • Une fois: Le périphérique de démarrage préféré spécifié sert uniquement à démarrer lors du prochain redémarrage du serveur. Une fois le serveur redémarré, la valeur du Support de démarrage préféré sera automatiquement remplacée par Aucun remplacement.
  • Permanent: Le paramètre du support de démarrage préféré prend effet de manière permanente.

Les valeurs par défaut sont Aucun remplacement et Une fois.

Priorité de démarrage

Lorsque le Support de démarrage préféré est défini sur Aucun remplacement, le système d'exploitation sera démarré conformément au paramètre de la Priorité de démarrage.

Vous pouvez cliquer sur pour déplacer une option vers le haut, puis sur pour déplacer une option vers le bas.

REMARQUE:
  • Les paramètres de priorité de démarrage sur le BIOS prennent effet immédiatement. Si le système d'exploitation est redémarré, les paramètres de priorité de démarrage sur le BMC seront synchronisés avec les paramètres du BIOS.
  • Les paramètres de priorité de démarrage sur le BMC prennent effet après le redémarrage du système d'exploitation. Après le système d'exploitation est redémarré, les paramètres de priorité de démarrage du BIOS seront synchronisées avec les paramètres du BMC.

Configuration du périphérique de démarrage

  1. Dans l'onglet Périphérique de démarrage , définissez le premier périphérique de démarrage pour le système d'exploitation du serveur. Pour plus de détails sur les paramètres, voir Tableau 1.
  2. Cliquez sur Enregistrer.
