Aperçu général

Présentation de la fonction

Sur la page Console virtuel, vous pouvez utiliser Console à distance avec HTML5 intégré ou Console à distance avec Java intégrée pour accéder au système d'exploitation du serveur.

Description du paramètre

Tableau 1 Console virtuel

Paramètre

Description

Console à distance avec HTML5 intégrée

La Console à distance avec HTML5 intégré offre deux modes d'accès :

  • La mode privée n'autorise qu'un seul utilisateur local ou un seul utilisateur VNC à accéder au serveur et à effectuer des opérations sur celui-ci via le BMC.
  • La mode partagée permet à deux utilisateurs locaux ou jusqu'à 5 utilisateurs VNC d'accéder et d'effectuer simultanément des opérations sur le serveur via le BMC. Chaque utilisateur peut visualiser les opérations effectuées par l'autre utilisateur.

Console à distance avec HTML5 intégré vous permet de :

  • Ajuster l'écran à vos préférences à l'aide des boutons flottants, des boutons de zoom d'écran, des boutons de souris et du curseur de Clarté d'image.
  • Régler le périphérique d'entrée à l'aide des touches de combinaison et des touches de disposition du clavier.
  • Contrôler le système d'exploitation du serveur à l'aide des boutons de contrôle d'alimentation et enregistrez les opérations effectuées sur le système d'exploitation à l'aide du bouton d'enregistrement vidéo.
  • Monter un fichier image ou un dossier local à l'aide des boutons DVD et FDD.

Console distante intégrée à Java

La Console à distance avec Java intégrée offre deux modes d'accès :

  • La mode privée n'autorise qu'un seul utilisateur local ou un seul utilisateur VNC à accéder au serveur et à effectuer des opérations sur celui-ci via le BMC.
  • La mode partagée permet à deux utilisateurs locaux ou jusqu'à 5 utilisateurs VNC d'accéder et d'effectuer simultanément des opérations sur le serveur via le BMC. Chaque utilisateur peut visualiser les opérations effectuées par l'autre utilisateur.

La Console à distance intégrée vous permet de :

  • Ajuster l'écran à vos préférences à l'aide des boutons flottants, des boutons de zoom d'écran, des boutons de souris et du curseur de Clarté d'image.
  • Régler le périphérique d'entrée à l'aide des touches de combinaison, des indicateurs du clavier et des touches de disposition.
  • Contrôler le système d'exploitation du serveur à l'aide des boutons de contrôle d'alimentation et enregistrez les opérations effectuées sur le système d'exploitation à l'aide du bouton d'enregistrement vidéo.
  • Monter un lecteur de DVD physique, un lecteur de disquettes, un fichier image ou un dossier local.
  • Créer un DVD ou un fichier image logiciel à l'aide du bouton de création d'image.

Environnement d'exécution

Tableau 2 répertorie les systèmes d'exploitation (OS), les navigateurs Web et l'environnement d'exécution Java (JRE) requis pour utiliser la console virtuelle à distance.

  • La Console à distance avec Java intégré nécessite le JRE (Java Runtime Environment). Si le JRE n'est pas installé, cliquez ici pour le télécharger à partir du site officiel AdoptOpenJDK. Si la Console à distance avec Java intégré ne peut pas être utilisée après l'installation du JRE, cliquez sur Plus d'informations.
  • Si la version TLS est réglée sur TLS 1.3 uniquement sur la page Utilisateur et sécurité > Gestion de la sécurité, l'environnement d'exécution BMC ne prend pas en charge les versions de navigateurs suivants :
    • Safari 11.0 à 12.0
    • Microsoft Edge 12 à 18

Ouverture de la console virtuelle à distance

Lors de la saisie du mot de passe du système d'exploitation ou du BIOS dans la Console à distance virtuelle :
  • Si le clavier utilisé est conforme aux paramètres du clavier du système d'exploitation, utilisez le clavier actuel.
  • Si le clavier utilisé n'est pas conforme aux paramètres du clavier du système d'exploitation, utilisez le clavier du système d'exploitation.
Si une alerte de sécurité de site Web est affichée, vous pouvez ignorer ce message ou effectuer l'une des actions suivantes pour bloquer cette alerte :
  • Importez un certificat de confiance et un certificat racine dans le BMC.
  • Si aucun certificat de confiance n'est disponible, ajoutez le BMC à la liste des sites d'exception du panneau de configuration Java. Cette opération présente toutefois des risques pour la sécurité.
  • Sur la page Accueil de l'interface utilisateur Web BMC, sélectionnez Console à distance avec Java intégré ou Console à distance avec HTML5 intégré dans la liste déroulante.

    Le mode partagé permet à deux utilisateurs d'utiliser simultanément la console à distance pour accéder et effectuer des opérations sur le serveur. Chaque utilisateur peut visualiser les opérations effectuées par l'autre utilisateur.

    Le mode privé permet à un seul utilisateur d'utiliser la console à distance pour accéder et effectuer des opérations sur le serveur. Si vous sélectionnez ce mode, la fonction de capture d'écran manuelle n'est pas disponible.

  • Ouvrez le navigateur et entrez ce qui suit dans la zone d'adresse :
    • L'un ou l'autre de ce qui suit :
      • Pour se connecter au Console à distance intégré HTML5, entrez :
        • https://IP address/remoteconsole?openWay=html5 ou https://IP address/remoteconsole?openway=html5
        • https://IP address/remote_access.asp?authParam=key&lp=lang&openWay=html5 ou https://IP address/remote_access.asp?authParam=key&lp=lang&openway=html5
      • Pour se connecter au Console à distance intégré Java, entrez :
        • https://IP address/remoteconsole, https://IP address/remoteconsole?openWay=jre, ou https://IP address/remoteconsole?openway=jre
        • https://IP address/remote_access.asp?authParam=key&lp=lang&openWay=jre ou https://IP address/remote_access.asp?authParam=key&lp=lang&openway=jre
      • key peut être réglée via l'interface Redfish pour une connexion directe avec le KVM.
      • lp indique le type de la langue utilisée sur la console.
      • openWay ne peut être qu'openway ou openWay. Toute autre valeur entraînera un passage à la console Java.
    • https://IP address/kvmvmm.asp
    • https://IP address/login.html?redirect_type=1

      IP address indique l'adresse IP du port réseau de gestion BMC.
