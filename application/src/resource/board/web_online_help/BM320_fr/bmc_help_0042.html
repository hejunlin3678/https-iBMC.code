Gestion de la langue

Présentation de la fonction

Sur la page Gestion de la langue, vous pouvez activer et désactiver la langue prise en charge par le BMC.

  • Seuls l'administrateur et l'utilisateur autorisé pour les paramètres communs peuvent activer ou désactiver les langues.
  • Actuellement, le BMC ne prend en charge que le chinois, l'anglais, le japonais, le russe et le français. Le chinois et l'anglais sont activés par défaut. Le japonais, le russe et le français peuvent être activés ou désactivés selon les besoins.

Description du paramètre

Tableau 1 Gestion de la langue

Paramètre

Description

ID

Numéro de séquence d'une langue.

Code de langue

Code d'une langue. Par exemple, en signifie Anglais, zh pour Chinois, ja pour Japonais, ru pour russien et fr pour Français.

Nom de la langue

Langue correspondant au code.

Opération

Active ou désactive la langue spécifiée.

  • La langue est activée.
  • La langue est désactivée.

Interroger les langues activées

  1. Choisissez Paramètres BMC dans le menu principal.
  2. Sélectionnez Gestion de la langue.

    Vous pouvez voir les langues activées.

Activation d'une langue

  1. Sur Gestion de la langue, localisez la langue que vous voulez activer et réglez l'icône sur .
  1. Cliquez sur Confirmation.

    « Opération réussie » s'affiche.
