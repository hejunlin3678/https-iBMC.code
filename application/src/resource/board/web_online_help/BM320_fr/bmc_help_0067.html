Gestion USB

Présentation de la fonction

La page Gestion USB permet d'interroger et de configurer le port de gestion d'BMC à connexion directe.

La fonction de gestion USB doit être utilisée avec le port de gestion BMC à connexion directe sur le serveur.

Description du paramètre

Tableau 1 Gestion USB

Catégorie

Description

Gestion USB
Une fois cette fonction activée, vous pouvez accéder à BMC par le port USB sur l'oreille de montage du serveur en utilisant :
  • PC local : Vous pouvez accéder à l'interface utilisateur Web BMC (adresse IP 169.254.1.5) en utilisant un navigateur ou accéder à l'interface CLI BMC en utilisant SSH depuis un PC local.
  • Appareil mobile : Vous pouvez utiliser l/application mobile SmartServer pour accéder à BMC depuis un appareil mobile.
REMARQUE:
  • Ce paramètre est activé par défaut.
  • Seuls les PC Windows 10, les téléphones Android prennent en charge cette fonction.

État du périphérique USB

État du périphérique USB.

  • Connecté : Un périphérique USB est accessible à partir du port USB.
  • Non connecté : Aucun appareil n'est accessible à partir du port USB.
