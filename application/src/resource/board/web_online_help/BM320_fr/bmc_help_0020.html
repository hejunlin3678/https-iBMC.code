Journaux système

Présentation de la fonction

Sur la page Journaux système, vous pouvez activer ou désactiver le port PCIe. Une fois le port PCIe activé, elle peut être utilisée pour la communication du système intrabande et hors bande et la collecte de données de la boîte noire. L'interface PCIe peut également être utilisée pour la communication MCTP si le MCTP est pris en charge.

Sur la page Journaux système, vous pouvez activer ou désactiver la boîte noire. Une fois la boîte noire activée, vous pouvez télécharger les données stockées dans la boîte noire vers un hôte local.

Une boîte noire se compose d'une mémoire et d'un logiciel de surveillance des erreurs.

  • La mémoire est une puce de stockage intégrée qui enregistre les informations de défaut, indépendamment des disques durs du serveur.

    La mémoire fournit une capacité de stockage maximale de 4 Mo et enregistre les informations du noyau lorsque le système d'exploitation tombe en panne.

  • Le logiciel de surveillance des erreurs enregistre les informations du noyau lorsque le système d'exploitation tombe en panne.

    Avant d'utiliser la fonction Boîte noire, assurez-vous que le logiciel de surveillance des erreurs (par exemple, BMA) a été installé sur le serveur. Pour plus de détails sur l'installation et l'utilisation de le BMA, consultez le Guide de l'utilisateur BMA.

  • Si le pilote de la boîte noire n'a pas été installé sur les serveurs après l'activation de la fonction de la boîte noire, des périphériques inconnus peuvent être détectés sur le système d'exploitation.

Sur la page Données du port série, vous pouvez activer ou désactiver la fonction d'enregistrement des données transmises sur le port série et télécharger les dernières données.

Procédure

Tableau 1 Opérations de boîte noire

Opération

Procédure

Activer ou désactiver le port PCIe.
  1. Pour activer le port PCIe, définissez Port PCIe sur . Pour désactiver le port PCIe, définissez Port PCIe sur . Cliquez sur ou pour modifier les paramètres.
  2. Redémarrez le serveur.
REMARQUE:
  • Le port PCIe est activé par défaut.
  • Si les paramètres du port PCIe sont modifiés, redémarrez le serveur pour que les paramètres prennent effet.

Télécharger les données de la boîte noire.

Assurez-vous que la fonction boîte noire est activée.

Dans la zone Boîte noire, cliquez sur .

Les données de la boîte noire seront automatiquement téléchargées dans le répertoire par défaut du PC local.
REMARQUE:
  • Assurez-vous que la fonction est activée par défaut.
  • Le BMC télécharge uniquement le fichier de données de la boîte noire du serveur vers le PC local, mais sans les analyser. Pour plus d'informations sur l'analyse d'un fichier de données de boîte noire, consultez le Guide de l'utilisateur BMA.
  • Les informations d'enregistrement de fichier affichées varient selon le navigateur utilisé.
Tableau 2 Opérations sur les données du port série

Opération

Procédure

Activer ou désactiver la fonction d'enregistrement des données de port série du système.
Pour activer cette fonction, définissez Données du port série sur . Pour désactiver cette fonction, définissez Données du port série sur . Cliquez sur ou pour modifier les paramètres.
REMARQUE:

La fonction de données du port série est activée par défaut.

Téléchargez les données du port série du système.

Téléchargez les données du port série uniquement lorsque le paramètre Données du port série est définie sur .

Cliquez sur dans la zone Données du port série.

Les données du port série du système seront automatiquement téléchargées dans le répertoire par défaut du PC local.

REMARQUE:
  • Les dernières données du port série du système sont téléchargées.
  • Les informations d'enregistrement de fichier affichées varient selon le navigateur utilisé.
