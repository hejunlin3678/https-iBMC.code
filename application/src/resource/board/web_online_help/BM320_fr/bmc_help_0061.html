Description du paramètre

Tableau 1 Others

Paramètre

Description

BBU
Fournit les informations sur l'unité de backup de batterie (BBU) (uniquement pour le serveur qui prend en charge les BBU):

  • Nombre maximal de BBU et nombre de BBU présentes

  • Nom, ID, version micrologiciel, capacité restante, état de fonctionnement, modèle de batterie, numéro de série et fabricant de chaque BBU

Carte CPU
Fournit des informations sur la carte CPU :
  • Nombre maximum et nombre réel de cartes CPU actuelles.
  • Nom, fabricant, numéro d'emplacement, type, version du circuit imprimé, version du CPLD, ID de la carte, UID du composant et puissance de chaque carte CPU.

Panneau arrière pour disque dur

Fournit des informations sur le fond de panier du disque dur :

  • Nombre maximum de fonds de panier de lecteurs extensibles et le nombre de fonds de panier de lecteurs en position.
  • Nom, emplacement, fabricant, numéro, type, version du circuit imprimé, version CPLD, ID de carte, UID du composant, numéro de pièce et numéro de sérié de chaque fond de panier du disque dur

Expander
Fournit des informations sur la carte expander (uniquement pour le serveur qui prend en charge les cartes expander):
  • Nombre maximum et nombre réel de cartes expander
  • Nom, ID de la carte, UID du composant, CPLD Version, nombre, description, emplacement, fabricant, version PCB, numéro de pièce et numéro de série de chaque carte expander

Fond de panier du ventilateur

Fournit des informations sur le fond de panier du ventilateur：

  • Nombre maximum et nombre réel de fond de panier du ventilateur
  • Nom, emplacement, fabricant, type, version PGB, ID de la carte et UID du composant de chaque fond de panier du ventilateur

Adaptateur M.2

Fournit des informations sur l'adaptateur M.2, y compris le nom, la description, le emplacement, la version PCB, l'ID de la carte, le numéro de pièce et le numéro de série (uniquement pour le serveur qui prend en charge les adaptateurs M.2).

Carte OCP
Fournit des informations sur la carte OCP (uniquement pour le serveur qui prend en charge les carte OCP) :
  • Nombre maximum de cartes OCP et nombre de cartes OCP existantes
  • Description, emplacement, fabricant, numéro d'emplacement, numéro d'identification du fournisseur, numéro d'identification de périphérique, numéro de sous-fournisseur, numéro de sous-périphérique et ressource connectée de chaque carte OCP.

Carte PCIe

Fournit des informations sur la carte PCIe :

  • Nombre maximum de cartes PCIe et nombre de cartes PCIe existantes
  • Description, emplacement, fabricant, numéro d'emplacement, ID du fournisseur, ID du périphérique, numéro de pièce, ID du sous-fournisseur, ID du sous-périphérique et ressource connectée de chaque carte PCIe.

Carte Riser

Fournit des informations sur la carte Riser :

  • Nombre maximum et nombre réel de carte riser
  • Nom, fabricant, emplacement, type, version du circuit imprimé, ID de carte, UID du composant, code de nomenclature BOM et numéro de sérié de chaque carte Riser

Module de sécurité

Fournit des informations sur le module de sécurité :

  • Nombre maximum et nombre réel de modules de sécurité
  • Type de spécification, version de spécification, fabricant, version de fabrication et état d'autotest de chaque module de sécurité
