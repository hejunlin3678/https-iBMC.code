Smart Provisioning Management

Function Description

Smart Provisioning is a tool used for intelligent server deployment.

On the Smart Provisioning Management page, you can perform the following operations:
  • Configure the function for device information collection.
  • Configure the function for booting OS from Smart Provisioning.

Parameter Description

Table 1 Smart Provisioning management

Parameter

Description

Collect Server Info

If this function is enabled, the in-band system accesses Smarting Provisioning to collect server information when the in-band system starts for the first time upon the power-on of the PSUs. Then, the server starts based on the settings of Preferred Boot Medium and Boot Priority in BIOS Settings.

It is disabled by default.

Boot OS from SP

After this function is enabled, the in-band system directly enters Smart Provisioning once the server is restarted.

It is disabled by default.

NOTE:

The setting takes effect only upon the next restart. This function is automatically disabled when the system enters Smart Provisioning.
