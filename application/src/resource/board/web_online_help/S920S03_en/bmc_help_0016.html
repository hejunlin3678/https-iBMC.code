Alarm & SEL

Function Description

On the Alarm & SEL page, you can:

  • Query active alarms of the server.
  • View and search for system events, and download and clear all system events.

Parameter Description

Table 1 Alarms and SEL

Parameter

Description

No.

Sequential number of an event.

Severity

Severity of an alarm.

  • : A critical alarm may power off the server, and even interrupt system services. You must take corrective actions immediately.
  • : A major alarm has a major impact on the system. It affects the normal operating of the system or may cause service interruption.
  • : A minor alarm has a minor impact on the system, but you need to take corrective actions as soon as possible to prevent a more severe alarm.
  • : An event occurred during the normal running of the system.

Object Type

Type of the component, for which the alarm was generated.

Status

Status of the system event.

Value:

  • Asserted: indicates that a system event is generated.
  • Deasserted: indicates that a system event is cleared.

Event Code

Uniquely identifies an alarm.

Generated

Date and time when the alarm was generated.

Description

Supplementary information about the alarm.

Handling Suggestion

Suggestions on how to clear the alarm.

Searching for System Events

  1. Choose Maintenance > Alarm & SEL, and select the System Events tab.
  2. Click Search Criteria.

    The page for setting the search criteria is displayed.

  3. Set search criteria. For details about the parameters, see Table 2.

    Table 2 Search criteria

    Parameter

    Description

    Alarm Severities

    Severity of the system event.

    Value: All, Critical, Major, Minor, or Informational

    Object Type

    Type of the component, for which the alarm was generated.

    Value: varies depending on the server model.

    Generated

    Date and time when the system event was generated.

    Value:

    • ALL
    • Today
    • Last 7 days
    • Last 30 days
    • Custom
    NOTE:

    If you select Custom, you must specify the start date and end date.

    Custom

    Customized search criteria.

    You can enter the following in the text box:

    • Event description
    • Complete event code, with or without 0x.

  4. Click Query.

    The events that meet the filter criteria are displayed.

Deleting All System Events

Deleted system events cannot be restored. Exercise caution when deleting system events.

  1. Choose Maintenance > Alarm & SEL, and select the System Events tab.
  2. Click at the upper right corner of the page.

    All system events will be cleared.

Downloading System Events

  1. Choose Maintenance > Alarm & SEL, and select the System Events tab.
  2. Click at the upper right corner of the page.

    The downloaded file is automatically saved to the default directory on the local PC.
