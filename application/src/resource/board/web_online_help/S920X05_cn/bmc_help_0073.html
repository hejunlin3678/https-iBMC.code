SSL证书更新

功能介绍

通过使用“SSL证书更新”界面的功能，您可以：
  • 设置SSL证书更新。
  • 配置CA服务器。
  • 自定义证书主题。

参数说明

表1 SSL证书更新

参数

描述

SSL证书更新

说明：
  • 当系统锁定模式关闭、用户具有安全配置权限和CA服务器完成配置时，支持SSL证书更新。
  • SSL证书更新功能需要同时导入服务器根证书和客户端证书，否则会导致认证失败。

自动更新使能

开启或关闭证书自动更新功能。

开启“自动更新使能”后，SSL证书支持自动刷新，自动刷新时间间隔为1小时。

默认为关闭状态。

证书更新

单击“证书更新”，更新SSL证书。

CA服务器配置

服务器地址

CA服务器的地址。

取值原则：IPv4地址、IPv6地址或域名，且最大长度为67个字符。

端口

CA服务器的端口号。

取值范围：1～65535。

路径

CA服务器的路径。

取值原则：长度为1～128个字符，以斜线（/）开头，由数字（0-9），字母（a-z，A-Z），下划线（_），斜线（/）组成。

证书主题

国家（C）

证书所属的国家。

取值原则：包含所有英文字母，且长度必须为两字符。

省份（S）

证书所属的省份。

取值原则：长度为0～128个字符，可由数字（0-9）、字母（a-z，A-Z）、中划线（-）、下划线（_）、点（.）、空格符号组成。

城市（L）

证书所属的城市。

取值原则：长度为0～128个字符，可由数字（0-9）、字母（a-z，A-Z）、中划线（-）、下划线（_）、点（.）、空格符号组成。

公司（O）

证书所属的公司。

取值原则：长度为0～64个字符，可由数字（0-9）、字母（a-z，A-Z）、中划线（-）、下划线（_）、点（.）、空格符号组成。

部门（OU）

证书所属的部门。

取值原则：长度为0～64个字符，可由数字（0-9）、字母（a-z，A-Z）、中划线（-）、下划线（_）、点（.）、空格符号组成。

常用名（CN）

证书所属的常用名。

取值原则：长度为1～64个字符，可由数字（0-9）、字母（a-z，A-Z）、中划线（-）、下划线（_）、点（.）、空格符号组成。

内部名（IN）

证书的内部名。

取值原则：长度为0～64个字符，可由数字（0-9）、字母（a-z，A-Z）、中划线（-）、下划线（_）、点（.）、空格符号组成。

邮件地址（E）

证书的邮件地址。

取值原则：长度为0～255个字符，符合邮箱地址规则。

服务器根证书

通过正式的CA证书颁发机构申请的服务器根证书。在建立数据连接时，使用此处上传的服务器根证书对CA服务器发送来的服务器证书报文进行“TLS”验证。

说明：
  • 支持上传服务器根证书文件的格式为“.crt”、“.cer”和“.pem”，*.cer格式的证书的编码为Base64，最大不超过100KB，且不支持上传相同的证书。
  • 请定期更新证书，否则可能存在安全风险。

客户端证书

通过正式的CA证书颁发机构申请的客户端证书。

说明：
  • 支持上传客户端证书格式为*.p12和*.pfx，最大不超过100KB。
  • 请定期更新证书，否则可能存在安全风险。

证书信息

显示上传的证书信息，包括：

  • 签发者
  • 使用者
  • 有效起止日期
  • 序列号
  • 证书吊销列表，状态为：
    • 已配置：表示该证书的吊销文件已上传，在TLS连接时，会进行证书吊销校验。
    • 未配置：表示该证书的吊销文件未上传。
说明：
  • 证书吊销文件的格式为“*.crl”，文件名不能为空，最大不超过100KB。
  • 吊销列表过期会导致相应的认证功能失败。
  • 客户端证书不支持证书吊销列表。

上传证书

  1. 单击“服务器根证书”或“客户端证书”后的"上传"。

    弹出本地文件选择器或者出现客户端证书上传弹窗。

    • 服务器根证书支持*.cer、*.crt、*.pem格式，*.cer格式的证书的编码为Base64，最大不超过100KB，且不支持上传相同的证书。
    • 客户端证书支持*.p12和*.pfx格式，最大不超过100KB。

  2. 选择正确格式文件。
  3. 单击“确定”。

配置证书吊销列表

证书吊销文件的格式为“*.crl”，文件名不能为空，最大不超过100KB。

  1. 从证书颁发机构获取证书吊销文件。
  2. 在“服务器证书信息”区域单击“证书吊销列表”后的“上传”。
  3. 选择证书吊销文件。
  4. 输入当前登录用户密码并单击“保存”。

删除证书吊销列表

删除证书吊销列表，可能会导致使用过期的证书，请注意安全风险。

  1. 单击指定证书对应的“证书吊销列表”的“删除CRL”。

    弹出操作确认对话框。

  2. 单击“确认”。
