Parameter

Description

CPU information

Provides the following information about each CPU installed in the server:

  • Number of processors in full configuration and number of installed processors
  • Name, manufacturer, model, CPU ID, clock speed, and serial number of each processor
  • Number of cores and threads
  • L1, L2, and L3 cache capacity
  • Processor status
  • Serial Number
  • Other parameters
