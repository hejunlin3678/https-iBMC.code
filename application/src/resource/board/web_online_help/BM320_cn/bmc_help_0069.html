表1 组件

参数

描述

基础计算组件
显示服务器基础计算组件信息。
  • 基础计算组件满配个数和当前在位个数。
  • 基础计算组件的基本信息：名称、位置、厂商、槽位、描述和UID。
  • 单击基础计算组件名称左侧的，显示基础计算组件的扩展信息：序列号、部件编码、PCB版本、CPLD版本和硬件自描述版本。
