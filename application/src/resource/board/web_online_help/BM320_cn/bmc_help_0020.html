系统日志

功能介绍

通过使用“PCIe接口”，您可以开启或关闭PCIe接口。开启该功能时，PCIe接口可以用于带内带外系统通信和黑匣子数据收集，如果支持MCTP功能，PCIe接口也可以用于MCTP通信。

通过使用“黑匣子功能”，您可以开启或关闭黑匣子数据下载功能，开启该功能时，您可以下载黑匣子存储器中的数据到本地。

黑匣子包含一个存储器和一款故障监控软件：

  • 黑匣子存储器是系统内置的用于故障信息记录的存储芯片。它不依赖于服务器的硬盘。

    黑匣子存储器的最大容量为4MB，用于记录操作系统崩溃时的内核信息。

  • 故障监控软件记录服务器操作系统崩溃时的内核信息。

    在使用黑匣子功能前，服务器上必须已安装黑匣子的故障监控软件（例如BMA，其安装和使用方法可参考BMA用户指南）。

  • 在开启黑匣子功能的情况下，如果服务器上未安装黑匣子驱动，则可能在OS侧出现未知设备。

通过使用“系统串口数据记录功能”区域框的功能，您可以开启或关闭串口数据下载记录功能，开启该功能时您可以下载系统串口最近的数据到本地。

操作步骤

表1 黑匣子功能操作步骤

操作

操作步骤

开启或关闭PCIe接口
  1. 将“PCIe接口”右侧的按钮设置为，表示开启PCIe接口。将按钮设置为，表示关闭PCIe接口。单击或，可切换状态。
  2. 重启服务器。
须知：
  • PCIe接口默认为开启状态。
  • 开启或关闭PCIe接口都需要重启服务器后才能生效。

下载黑匣子数据文件

请在“黑匣子功能”为状态下下载黑匣子数据文件。

单击“黑匣子功能”区域框的。

黑匣子数据文件将自动保存到本地PC的默认地址。

说明：
  • “黑匣子功能”默认为开启状态。
  • BMC不提供黑匣子数据文件的解析功能。关于黑匣子数据文件的解析功能请参考BMA用户指南。
  • 在不同浏览器下，页面提示保存文件的信息略有不同。
表2 系统串口数据记录功能操作步骤

操作

操作步骤

开启或关闭系统串口数据记录功能

将“系统串口数据记录功能”右侧的按钮设置为，表示开启系统串口数据记录功能。将按钮设置为，表示关闭系统串口数据记录功能。单击或，可切换状态。

说明：

“系统串口数据记录功能”默认为开启状态。

下载系统串口数据文件

请在“系统串口数据记录功能”为状态下下载系统串口数据文件。

单击“系统串口数据记录功能”区域框的。

系统串口数据文件自动保存到本地PC的默认路径。

说明：
  • 下载的数据文件为每个系统串口最近的数据。
  • 在不同浏览器下，页面提示保存文件的信息略有不同。
