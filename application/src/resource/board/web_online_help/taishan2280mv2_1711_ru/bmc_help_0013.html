Настройки BIOS

Описание функций

На странице Настройки BIOS установите первое загрузочное устройство для ОС сервера.

Описание параметров

Табл. 1 Загрузочное устройство

Категория

Описание

Приоритетный носитель для загрузки
  • Диск: загрузка ОС осуществляется с жесткого диска.
  • CD/DVD: загрузка ОС осуществляется с CD-диска или DVD-диска.
  • FDD/Съемное устройство: загрузка ОС осуществляется с виртуального флоппи-диска (FDD) или съемного устройства.
  • PXE: загрузка ОС осуществляется из среды PXE (Preboot Execution Environment).
  • Настройки BIOS: отображение меню настроек BIOS при запуске сервера.
  • Не настроено: выберите эту опцию для загрузки ОС с первого загрузочного устройства, указанного в BIOS.
  • Один раз: приоритетный носитель для загрузки используется только для загрузки при следующем перезапуске сервера. После перезапуска сервера значение Приоритетный носитель для загрузки автоматически изменяется на Без перезаписи.
  • Всегда: Настройка приоритетного носителя для загрузки имеет силу Всегда.

Значения по умолчанию — Не настроено и Один раз.

Приоритет загрузки

Когда Приоритетный носитель для загрузки установлен в значение Безперезаписи, ОС загружается в соответствии с настройкой Приоритет загрузки.

Нажмите , чтобы переместить опцию вверх, и , чтобы переместить опцию вниз.

ПРИМЕЧАНИЕ:
  • Настройки приоритета загрузки в BIOS вступают в силу немедленно. При перезапуске ОС настройки приоритета загрузки на BMC будут синхронизированы с настройками в BIOS.
  • Настройки приоритета загрузки на BMC вступают в силу после перезапуска ОС. После перезапуска ОС настройки приоритета загрузки в BIOS будут синхронизированы с настройками на BMC.

Настройка загрузочного устройства

  1. На вкладке Загрузочное устройство укажите устройство, с которого в первую очередь будет осуществляться загрузка ОС сервера. Более подробная информация о параметрах приводится в Табл. 1.
  2. Нажмите Сохранить.
