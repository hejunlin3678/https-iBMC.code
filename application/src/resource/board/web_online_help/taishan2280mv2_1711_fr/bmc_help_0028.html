Utilisateurs en ligne

Présentation de la fonction

La page Utilisateurs en ligne vous permet d'effectuer les opérations suivantes :

  • Voir les utilisateurs qui se sont connectés à le BMC.
  • Forcer la déconnexion des utilisateurs en ligne.

Seul un administrateur peut forcer la déconnexion des utilisateurs.

Description du paramètre

Tableau 1 Utilisateurs en ligne

Paramètre

Description

N°

Numéro de séquence d'un utilisateur connecté.

Nom d'utilisateur

Nom de l'utilisateur qui s'est connecté à le BMC ou au KVM.

Méthode de connexion

Mode permettant à un utilisateur de se connecter.

Les options sont les suivantes :

  • GUI(SSO): un utilisateur s'est connecté à WebUI BMC en mode SSO.
  • GUI: un utilisateur s'est connecté à WebUI BMC en mode non-SSO.
  • CLI: Un utilisateur s'est connecté à le BMC par l'interface CLI (Command-Line Interface).
  • KVM: Un utilisateur s'est connecté au système d'exploitation via la console virtuelle à distance.
  • Redfish: Un utilisateur s'est connecté à le BMC via l'interface Redfish.
  • VNC: Un utilisateur se connecte au système d'exploitation via la console de réseau virtuel.

Adresse IP

Adresse IP pour la connexion et se connecter à le BMC.

Value: IP address ou COM

REMARQUE:

La valeur COM indique que l'utilisateur se connecte à le BMC via le port série.

Temps de connexion

Heure à laquelle l'utilisateur s'est connecté à BMC.

Opération

Forcer la déconnexion d'un utilisateur.

Cliquez sur Déconnexion pour déconnecter de force l'utilisateur.
