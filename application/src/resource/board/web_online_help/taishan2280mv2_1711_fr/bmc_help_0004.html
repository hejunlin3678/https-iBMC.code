Opérations de base

Tableau 1 décrit les opérations de base de l'interface utilisateur Web BMC.

Tableau 1 Opérations de base

Operation

Procedure

Sélectionnez une langue.

Déplacez la souris sur dans le coin supérieur droit de l'interface utilisateur Web et sélectionnez la langue à utiliser dans la liste déroulante.

Voyez les informations de base du serveur.

Choisissez Accueil > Plus de détails > Info système.

La page Info système affiche les informations de base du serveur, y compris les informations sur le produit et les informations sur les processeurs, la mémoire, le adaptateurs réseau, les capteurs et les autres composants.

Obtenez des informations d'aide en ligne.

Cliquez sur sur la page BMC WebUI.

Interrogez les informations utilisateur.

Après vous être connecté à le BMC, cliquez sur le nom d'utilisateur (par exemple, test) en regard de dans le coin supérieur droit.

Les informations relatives à l'utilisateur sont affichées, notamment le nom d'utilisateur, le rôle, l'adresse IP et le temps de connexion.

Déconnectez-vous de l'interface utilisateur Web BMC

Cliquez sur le nom d'utilisateur dans le coin supérieur droit de la page et choisissez Déconnecter dans le menu déroulant.

Mettez sous tension ou hors tension le système d'exploitation du serveur.

Cliquez sur

pour la mise sous tension ou hors tension du système d'exploitation du serveur.

Le vert indique que le système d'exploitation du serveur est allumé. Le gris indique que le système d'exploitation du serveur est éteint.

Définissez le code UID sur le serveur.

L'indicateur UID indique l'emplacement du serveur dans un châssis.

Déplacez le curseur dans le coin supérieur droit de l'écran et sélectionnez Activé, Désactivé, ou Clignotant.

Si vous sélectionnez Clignotant, le voyant UID clignote pendant 255 secondes.

Affichez la gravité et la quantité d'alarmes actives du serveur.

Vous pouvez cliquer sur la gravité ou la quantité pour passer à Entretien > Alarme et SEL > Alarme en cours.

  • : Une alarme critique peut mettre le serveur hors tension et peut même interrompre des services système. Vous devez prendre des mesures correctives immédiatement.
  • : Une alarme majeure a un impact majeur sur le système. Cela affecte le fonctionnement normal du système ou peut entraîner une interruption de service.
  • : Une alarme mineure a un impact mineur sur le système, mais vous devez prendre des mesures correctives dès que possible pour éviter une alarme plus grave.
