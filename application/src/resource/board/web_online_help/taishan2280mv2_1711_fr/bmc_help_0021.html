Journaux BMC

Présentation de la fonction

  • Sur la page Journaux des opérations, vous pouvez voir et télécharger les journaux enregistrés pendant le fonctionnement du système, y compris des informations sur le démarrage du système, la transition d'état et la configuration effectuée par les utilisateurs sur le BMC.

    Le BMC offre une capacité de 200 Ko pour stocker jusqu'à 2000 enregistrements de journaux des opérations.

    Lorsque les journaux des opérations atteignent 200 Ko, ils seront automatiquement compressés. Lorsqu'un nouveau paquet compressé est généré, l'ancien paquet compressé sera automatiquement supprimé.

    Les journaux des opérations qui enregistrent les opérations de mise sous tension, de mise hors tension et de réinitialisation indiquent que le BMC a correctement déclenché les opérations, mais ne signifie pas nécessairement que ces opérations ont été exécutées avec succès sur le matériel.

  • Sur la page Exécution de journaux, vous pouvez voir les journaux RAS du serveur.

    Le BMC offre une capacité de 200 Ko pour stocker jusqu'à 2000 enregistrements des journaux d'exécution.

    Lorsque les journaux atteignent 200 Ko, ils seront automatiquement compressés. Lorsqu'un nouveau paquet compressé est généré, l'ancien paquet compressé sera automatiquement supprimé.

  • Sur la page Journaux de sécurité, vous pouvez effectuer les opérations suivantes :
    • Voir les journaux des connexions et déconnexions BMC via un port série Secure Shell (SSH) ainsi que les journaux des opérations de configuration.
    • Voir les journaux des opérations de recherche et de configuration effectuées sur SNMP.
    • Télécharger les journaux de sécurité.

    Le BMC offre une capacité de 200 Ko pour stocker jusqu'à 2000 enregistrements de journaux de sécurité.

    Lorsque les journaux atteignent 200 Ko, ils seront automatiquement compressés. Lorsqu'un nouveau paquet compressé est généré, l'ancien paquet compressé sera automatiquement supprimé.

Description du paramètre

Tableau 1 Journaux des opérations

Paramètre

Description

N°

ID d'une opération. La dernière opération est affichée en premier.

Interface

Interface sur laquelle l'opération a été effectuée.

Utilisateur

Utilisateur qui a effectué l'opération.

La valeur Utilisateur est affichée en tant que N/A (le nom d'utilisateur n'est pas affiché) dans l'un des scénarios suivants :

  • Le bouton d'identification de l'utilisateur (UID) ou le bouton d'alimentation est enfoncé.
  • L'interface est SNMPv1 ou SNMPv2c.
  • L'interface est IPMI et l'adresse IP est HOST. (Le journal enregistre un message IPMI envoyé par le système de service.)
  • L'adresse IP BMC et le mot de passe de l'utilisateur par défaut ont été modifiés à l'aide d'un cavalier.
  • Un composant a été échangé à chaud.

Adresse IP

Adresse IP à partir de laquelle l'opération a été effectuée.

  • La valeur HOST indique que l'opération a été déclenchée par le système de service.
  • L'adresse IP est au format X.X.X.X et indique que l'opération est effectuée par un client ayant accédé à le BMC.
  • La valeur 127.0.0.1 indique que l'opération a été déclenchée par l'hôte local dans l'un des scénarios suivants:
    • Le bouton UID, le bouton de mémoire riser ou le bouton de mise sous tension est enfoncé.
    • Le BMC est accessible via le port série local.
    • L'adresse IP BMC et le mot de passe de l'utilisateur par défaut ont été modifiés à l'aide d'un cavalier.
    • Un composant a été échangé à chaud.

Créé

Heure à laquelle l'opération a été effectuée.

Plus d'infos

Détails sur l'opération.

Si BMC redémarre après une mise à niveau à l'aide de l'interface utilisateur Web BMC, de l'interface de ligne de commande ou de l'interface IPMI, l'opération est enregistrée dans le format suivant :

  • Interface : N/A
  • Utilisateur : N/A
  • Adresse IP : 127.0.0.1
  • Détails : Reset BMC caused by upgrade successfully

Télécharger

Cliquez sur . Le fichier journal d'opération sera automatiquement téléchargé dans le répertoire par défaut du PC local.

Remarque : Si les valeurs Utilisateur et Adresse IP ne peuvent pas être analysées, elles sont affichées comme Inconnue.
