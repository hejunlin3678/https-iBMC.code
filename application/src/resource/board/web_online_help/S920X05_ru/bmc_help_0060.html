Табл. 1 Датчик

Категория

Описание

Датчики

Логический модуль или физическое устройство, которое осуществляет мониторинг индикаторов сервера.

Текущее значение

Текущее значение индикатора.

--: датчик не собрал действительные данные.

Статус

Статус порогового датчика.

  • OK: датчик работает исправно.
  • --: состояние датчика неизвестно.
  • NC: датчик определяет незначительный сбой.
  • CR: датчик определяет серьезный сбой.
  • NR: датчик определяет критический сбой.

Нижний порог критической тревоги

Нижнее пороговое значение, установленное для генерирования датчиком сигнала тревоги критического уровня.

--: это пороговое значение не поддерживается.
