Знакомство со справочником BMC

Решение BMC (Intelligent Baseboard Management Controller) предназначено для управления сервером и мониторинга его состояния работоспособности. BMC имеет следующие особенности:

  • Оптимальный пользовательский опыт

    BMC предоставляет удобный графический интерфейс пользователя (Графический интерфейс пользователя (GUI), Graphical User Interface), который позволяет упростить эксплуатацию и техническое обслуживание сервера (O&M).

  • Эффективное управление и техобслуживание

    BMC обеспечивает удаленное управление сервером и мониторинг работоспособности его компонентов.

  • Безопасный доступ к системе

    BMC оснащен различными механизмами для обеспечения безопасного доступа к интерфейсам управления.

Справка BMC содержит информацию по использованию WebUI BMC. Единицы измерения терабайт (ТБ), гигабайт (ГБ), мегабайт (МБ), килобайт (КБ) или байт (Б) преобразуются в десятичном формате 1024.

Для упрощения эксплуатации и техобслуживания сервера BMC предоставляет WebUI и интерфейс командной строки (CLI).

Клиенты BMC

Клиенты, используемые для входа в WebUI BMC, должны соответствовать требованиям, перечисленным в Табл. 1. Если необходимо использовать удаленную консоль, используйте браузер и Java правильной версии.

Для обеспечения просмотра полной страницы BMC WebUI используйте следующие разрешения экрана:

  • 1280 * 800

  • 1366 * 768

  • 1440 * 900

  • 1600 * 900

  • 1600 * 1200

  • 1680 * 1050

  • 1920 * 1080

  • 1920 * 1200

Если для версии TLS установлено значение Только TLS 1.3 на странице Пользователи и безопасность > Управление безопасностью, рабочая среда BMC не поддерживает следующие версии браузера:

  • Safari 11.0-12.0

  • Microsoft Edge 12–18
