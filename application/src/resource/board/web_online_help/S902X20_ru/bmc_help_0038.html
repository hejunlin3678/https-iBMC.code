Настройки сети

Описание функций

На странице Настройки сети можно просмотреть и настроить следующие параметры BMC:

  • Имя хоста
  • Сетевой порт
  • Версию IP-протокола и IP-адреса
  • Информация о DNS
  • VLAN
  • LLDP

Изменение IP-адреса сетевого порта управления может стать причиной сбоя сетевого соединения. Поэтому рекомендуется изменять IP-адрес только в случае крайней необходимости.

Описание параметров

Табл. 1 Настройки сети

Параметр

Описание

Имя хоста BMC

Название хоста BMC.

Значение: строка длиной от 1 до 64 символов.

Имя хоста может включать буквы, цифры и дефис (-), но не может начинаться или заканчиваться с дефиса.

Режим

Тип сетевого порта управления сервера (сетевого порта BMC).

Значение по умолчанию: Фиксированный

Фиксированный

При выборе данной опции необходимо указать сетевой порт BMC.

  • Выделенный порт: специально выделенный BMC (порт управления сервера).
  • Внешний порт PCIe: порт на плате PCIe (плата PCIe, поддерживающая NC-SI, к которой подключается кабель NC-SI).
  • Порт OCP: порт на плате OCP1.
  • Порт OCP2: порт на плате OCP2.

Значение по умолчанию — Выделенный порт.

Авто

После включения данной опции BMC автоматически выбирает сетевой порт BMC на основании статуса порта.

Необходимо настроить диапазон портов для автоматического выбора. При выборе BMC порта из нескольких портов используется следующий приоритет:

выделенный порт > порт PCIe (порты 1–2 или порты 1–4) > порт OCP (порты 1–2 или порты 1–4) > порт OCP2 (порты 1–2 или порты 1–4)

Порты OCP2 и порты PCIe являются взаимоисключающими. Если к плате NIC PCIe подключается кабель NC-SI, то порты NIC PCIe можно использовать для доступа к BMC, а порты OCP2 — нет. Если к плате NIC PCIe не подключается кабель NC-SI, то порты OCP2 можно использовать для доступа к BMC, а порты NIC PCIe — нет.

ПРИМЕЧАНИЕ:
  • Если в качестве сетевого порта BMC выбран сетевой порт на плате PCIe, можно использовать только плату PCIe, подключенную к кабелям NC-SI.
  • При выборе порта PCIe, OCP или OCP2 вручную или автоматически один и тот же физический порт выполняет функции управляющего порта и сервисного сетевого порта. Если в режиме Фиксированный или Авто используется порт PCIe, порт OCP или порт OCP2, то в целях обеспечения безопасности настройте VLAN для сетевого порта управления.
  • Если в качестве сетевого порта управления BMC выбран сетевой порт, то рядом с сетевым портом будет отображаться .

Режим NCSI

Режим переключения сетевого порта NCSI, если параметр Режим имеет значение Авто.

Значением по умолчанию — Ручное переключение.

Автоматическое переключение на резервный ресурс

Система автоматически переключается на внешний сетевой порт с более высоким приоритетом в зависимости от состояния подключения порта. На основании правила автоматического выбора BMC выбирает внешний сетевой порт управления в следующей последовательности:

порт 1 > порт 2 или порт 1 > порт 2 > порт 3 > порт 4.

Ручное переключение

Ручное переключение на внешний сетевой порт с более высоким приоритетом.

Сетевой порт BMC

Если параметр Режим имеет значение Фиксированный, необходимо указать сетевой порт управления.

Если параметр Режим имеет значение Авто, необходимо указать сетевые порты для автоматического выбора.

Сетевые протоколы

Поддерживаемые протоколы:

  • IPv4: только IPv4. При выборе данной опции можно конфигурировать только адреса IPv4.
  • IPv6: только IPv6. При выборе данной опции можно конфигурировать только адреса IPv6.
  • IPv4/IPv6: IPv4 и IPv6. При выборе данной опции можно конфигурировать адреса IPv4 и IPv6.

Значение по умолчанию: IPv4/IPv6

IPv4
  • Получать IP-адрес автоматически: автоматическое назначение адреса IPv4 для сетевого порта BMC.
  • Настроить IP-адрес вручную: настройка адреса IPv4 для сетевого порта BMC вручную.

    Настройки адреса IPv4 включают следующие параметры: IP-адрес, Маска подсети, Шлюз по умолчанию и MAC-адрес.

Значение по умолчанию — Настроить IP-адрес вручную.

ПРИМЕЧАНИЕ:
  • MAC-адрес – физический адрес платы NIC.
  • Если вы не используете шлюз по умолчанию, установите шлюз на любой IP-адрес в сетевом сегменте статического и плавающего IP-адресов.

IPv6
  • Получать IP-адрес автоматически: автоматическое назначение адреса IPv6 для сетевого порта BMC.
  • Настроить IP-адрес вручную: настройка адреса IPv6 для сетевого порта BMC вручную.

    Настройки адреса IPv6 включают следующие параметры: IP-адрес, Префикс, Шлюз по умолчанию, Локальный адрес канала и IP-адрес2.

Значение по умолчанию — Настроить IP-адрес вручную.

ПРИМЕЧАНИЕ:
  • Локальный адрес канала используется для локальной связи.
  • IP-адрес2 поддерживает максимум 15 адресов IPv6 при использовании автоматической настройки адресов без отслеживания состояния (SLAAC, Stateless Address AutoConfiguration).
  • Если вы не используете шлюз по умолчанию, установите шлюз на любой IP-адрес в сетевом сегменте статического и плавающего IP-адресов.

DNS
  • Автоматически получать адрес IPv4 DNS: автоматическое назначение адреса IPv4 для сервера DNS.
  • Автоматически получать адрес IPv6 DNS: автоматическое назначение адреса IPv6 для сервера DNS.
  • Настроить IP-адрес вручную: настройка информации DNS вручную. Настройки адреса DNS включают следующие параметры: Домен, Основной сервер и Альтернативный сервер.

Значение по умолчанию — Настроить IP-адрес вручную.

УВЕДОМЛЕНИЕ:
  • При автоматическом получении IP-адреса сетевого порта управления BMC информация о DNS может быть получена автоматически или вручную. Рекомендуется выбрать автоматическое получение.
  • При выборе ручной настройки IP-адреса сетевого порта управления BMC информация о DNS также вводится вручную.

Домен: имя домена для сервера.

Имя домена должно соответствовать следующим требованиям:

  • Строка длиной до 67 символов.
  • Имя домена может состоять только из циф, букв, дефисов (-) и точек (.).
  • Имя домена не может начинаться с дефиса (-) или точки (.), либо заканчиваться дефисом (-).
  • Имя домена может состоять из не более 63 символов между любыми двумя точками.

Основной сервер: IP-адрес основного сервера DNS.

Значение: адрес IPv4 или IPv6 или пустое значение.
  • Альтернативный сервер 1: IP-адрес первого альтернативного сервера DNS.
  • Альтернативный сервер 2: IP-адрес второго альтернативного сервера DNS.

Значение: адрес IPv4 или IPv6 или пустое значение.

VLAN

Настройка VLAN, к которой принадлежит сетевой порт NC-SI.

Значение по умолчанию: Отключить

ПРИМЕЧАНИЕ:
  • Для изоляции сервисной сети от управляющей сети рекомендуется включить VLAN и настроить VLAN ID.
  • Если для сетевого порта управления BMC выбрано значение Выделенный порт, настройки VLAN становятся недействительными. Если для сетевого порта управления BMC выбрано любое другое значение, кроме Выделенный порт, настройки VLAN действительны.

VLAN ID

VLAN, к которой принадлежит сетевой порт NC-SI.

Значение: целое число от 1 до 4094.

ПРИМЕЧАНИЕ:

После сохранения VLAN ID настройки вступят в силу через несколько секунд.

LLDP

ПРИМЕЧАНИЕ:

LLDP поддерживается только в том случае, если в качестве сетевого порта управления BMC используется выделенный сетевой порт.

LLDP

После включения LLDP BMC может отправлять MAC-адрес устройства непосредственно подключенным устройствам в стандартных пакетах, что позволяет NMS запрашивать и определять состояние связи по каналу.

Значение по умолчанию: Отключено

Режим работы

Режим работы должен быть Transmit.

BMC поддерживает только передачу пакетов LLDP и не может принимать пакеты LLDP.

Задержка передачи (с)

В текущем режиме работы передается пакет LLDP для уведомления соседних устройств об изменении локальной конфигурации BMC (изменение сетевого порта управления, подключение или отключение сетевого кабеля сетевого порта управления).

Задержка передачи (в секундах) используется для предотвращения отправки большого количества пакетов LLDP из-за частых изменений локальной конфигурации. Если локальная конфигурация BMC изменяется в течение периода задержки, система сбрасывает счетчик. Следующий пакет LLDP будет отправлен только по истечении задержки.

Значение: от 1 до 8192

Значение по умолчанию: 2

Интервал передачи (с)

Интервал (в секундах), с которым отправляются пакеты LLDP, если информация о локальной конфигурации не изменена.

Значение: от 5 до 32768

Значение по умолчанию: 30

Множитель времени удержания

Если соседний узел не получает пакет LLDP от BMC в течение указанного периода времени (интервал передачи * множитель времени удержания), соседний узел автоматически очищает ранее зарезервированные пакеты.

Значение: от 2 до 10

Значение по умолчанию: 4
