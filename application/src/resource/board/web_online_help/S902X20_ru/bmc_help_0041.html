Обновление настроек

Описание функций

На странице Обновление настроек можно выполнить импорт и экспорт файлов конфигурации BMC, BIOS и RAID-контроллеров.

  • Получать информацию и экспортировать файлы конфигурации может только пользователь с правами администратора. Файл конфигурации, который необходимо импортировать или экспортировать, сохраняется в формате .xml. Размер не может превышать 1 МБ.
  • Невозможно выполнить импорт настроек шифрования KVM, если функция KVM включена. На другие функции такого ограничения нет.
  • Настройки RAID-контроллера вступают в силу только после завершения самотестирования при запуске системы (POST, Power-On Self Test).
  • Если импортированные элементы конфигурации включают изменения версии TLS или сетевой конфигурации, то произойдет разъединение установленных интернет-соединений. При появлении на экране сообщения «Сбой импорта.» необходимо выполнить повторный вход в WebUI BMC и проверить журнал операций, чтобы определить причину.
  • Если в экспортированном файле конфигурации содержится конфигурация в шифрованном тексте по умолчанию, данная конфигурация не вступает в силу при импорте файла конфигурации на другой сервер. Чтобы конфигурация вступила в силу, необходимо изменить конфигурацию простым текстом, удалить теги комментариев из строки, а затем импортировать файл конфигурации.
  • В экспортированном файле конфигурации IP-адрес сетевого порта управления BMC отображается звездочками.
  • Конфигурация BMC, конфигурация BIOS и частичная конфигурация RAID-контроллера могут быть импортированы и экспортированы.

Импорт файла конфигурации

Импорт конфигурационных файлов сопряжен с высоким риском. Соблюдайте осторожность при выполнении данной операции.

  1. (Опционально) Измените файл конфигурации.

    1. Откройте файл конфигурации с помощью текстового редактора и найдите конфигурацию, которую необходимо отредактировать.
    2. Добавьте используемый пароль.

      Например, измените значение SenderName с ****** на Sendertext123456789012@xxx.com, как показано на Рис. 1.

      Рис. 1 Файл конфигурации до редактирования
    1. Удалите "<!--" в начале строки и "-->" в конце строки.
      Рис. 2 Файл конфигурации после редактирования
    1. Сохраните изменения и закройте файл конфигурации.

  2. Нажмите Выбрать файл и выберите файл конфигурации, который собираетесь выгрузить.

    Выгружаемый файл отобразится на странице.

    Файл конфигурации, который необходимо импортировать, сохраняется в формате .xml. Размер не может превышать 1 МБ.

  3. Нажмите Импорт.

    Откроется диалоговое окно с просьбой ввести текущий пароль пользователя.

  4. Введите текущий пароль пользователя и нажмите Да.

    На экране отобразится следующее сообщение:

    Новые настройки вступят в силу после перезагрузки BMC.
    • Конфигурация BMC и конфигурация RAID-контроллера вступают в силу сразу после их импорта.
    • После импорта конфигурации BIOS необходимо перезапустить ОС сервера, чтобы конфигурация вступила в силу.
      • Если вы выберете Перезагрузить позже, вы можете перезапустить ОС сервера в нужное время.
      • Если вы выберете Перезагрузить сейчас, появится страница Управление энергопотреблением. Можно выбрать нужный способ перезагрузки ОС сервера.

    Из конфигурации RAID-контроллера можно импортировать только конфигурацию Экстренное копирование, Экстренное копирование при ошибке SMART и JBOD. Конфигурация логических и физических дисков не может быть импортирована.

Экспорт файла конфигурации

  1. На странице Обновление настроек нажмите Экспорт.

    Файл конфигурации будет экспортирован и сохранен в каталог по умолчанию на локальном ПК.
