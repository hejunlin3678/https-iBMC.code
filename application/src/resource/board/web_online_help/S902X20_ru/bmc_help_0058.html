Описание

Основная информация

Информация о памяти включает следующее:

  • Максимальное и фактическое количество DIMM
  • Имя, производитель, объем, настроенная скорость, максимальная скорость, тип и местоположение каждого модуля DIMM

Подробная информация

После нажатия слева от имени модуля DIMM отобразится следующая информация:

Номер компонента, BOM-код, серийный номер, ширина битового поля, количество рангов, минимальное напряжение, сведения о типах и номер оригинального компонента.
