Диагностика

Описание функций

На странице Диагностика можно включить или выключить порт PCIe. После включения порта PCIe его можно использовать для связи MCTP, внутриполосной и внеполосной системной связи, а также для сбора данных «черного ящика».

На странице Диагностика можно включить или выключить «черный ящик». После его включения можно загрузить данные, хранящиеся в «черном ящике», на локальный хост.

Черный ящик состоит из памяти и ПО мониторинга неисправностей.

  • Память представляет собой встроенный чип, который записывает информацию о неисправностях независимо от жестких дисков сервера.

    Объем памяти составляет не более 4 МБ. При сбое ОС в эту память сохраняются данные ядра.

  • ПО мониторинга неисправностей записывает данные ядра при сбое ОС.

    Перед использованием функции черного ящика убедитесь, что ПО мониторинга неисправностей (например, BMA) установлено на сервере. Более подробная информация по установке и использованию BMA приводится в Руководстве пользователя BMA.

  • Если драйвер черного ящика не был установлен на серверах после включения функции черного ящика, ОС может обнаруживать неизвестные устройства.

На странице Диагностика в разделе Данные последовательного порта можно включить или отключить функцию записи данных, передаваемых через последовательный порт, а также загрузить последние 4 МБ данных.

  1. Для включения функции порта PCIe переведите переключатель Порт PCIe в положение . Для отключения функции порта PCIe переведите переключатель Порт PCIe в положение . Нажмите или , чтобы изменить настройки.

Загрузка данных черного ящика

Убедитесь, что функция черного ящика включена.

В разделе Черный ящик нажмите .

Данные черного ящика будут автоматически загружаться в каталог по умолчанию локального ПК.
ПРИМЕЧАНИЕ:
  • Черный ящик включен по умолчанию.
  • BMC поддерживает только загрузку файла данных черного ящика с сервера на локальный ПК, без анализа содержимого файла. Более подробная информация об анализе файла данных черного ящика приводится в руководстве по установке сервера.
  • Отображаемая информация о сохраненных файлах может отличаться в разных браузерах.

Загрузка системных данных последовательного порта

Переведите переключатель Данные последовательного порта в положение .

Нажмите в разделе Данные последовательного порта.

Данные, переданные через последовательный порт, автоматически загрузятся в каталог по умолчанию локального ПК.

ПРИМЕЧАНИЕ:
  • Системе доступны последние 4 МБ данных, переданных через последовательных порт.

  • Отображаемая информация о сохраненных файлах может отличаться в разных браузерах.
