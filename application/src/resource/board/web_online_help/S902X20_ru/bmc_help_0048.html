Не удалось открыть интегрированную удаленную консоль Java

Признак неисправности

Признак

Возможные причины

Интегрированная удаленная консоль Java не может быть открыта.
  • Версия JRE является некорректной.
  • Версия JRE несовместима с BMC.

Решение

  1. Проверьте, правильно ли установлена JRE.

    BMC поддерживает AdoptOpenJDK 8 JRE и AdoptOpenJDK 11 JRE.

    • Если установлена правильная версия JRE, обратитесь в службу технической поддержки. Если установлена правильная версия JRE, обратитесь в службу технической поддержки.
    • Если установлена неправильная версия JRE, выполните 2.

  2. Загрузите двоичный пакет JRE, соответствующий клиентской ОС, с официального сайта AdoptOpenJDK.
  3. Установите AdoptOpenJDK.

    • После распаковки пакета вручную настройте переменные среды JAVA_HOME и PATH.
    • Загрузите IcedTea Web с официального веб-сайта AdoptOpenJDK, распакуйте его и установите папку bin в переменную среды PATH.

  4. Выполните вход в BMC WebUI, в поле Виртуальная консоль на странице Главная выберите Запуск > Встроенная удаленная консоль Java (приватная) или Встроенная удаленная консоль Java (совместная).

    В ходе данного процесса автоматически загружается файл .jnlp.

  5. Откройте файл .jnlp.

    Файл .jnlp используется с ограничением по времени. Если он станет недействительным, обратитесь к пункту 4 и загрузите его снова.

    • Если клиент использует интерфейс командной строки (CLI) Linux или Windows, перейдите к каталогу, в котором находится .jnlp, и запустите javaws kvm.jnlp.
    • Если клиент использует Графический интерфейс пользователя (GUI), найдите файл .jnlp, щелкните по имени файла правой кнопкой мыши и выберите javaws. Если javaws недоступен в меню быстрого доступа, перейдите к папке bin в каталоге установки IcedTea Web.

Исходная тема: Устранение неполадок удаленной виртуальной консоли
