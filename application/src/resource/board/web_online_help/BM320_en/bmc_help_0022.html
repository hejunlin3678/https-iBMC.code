Work Records

Function Description

On the Work Records page, you can add and view work records.

  • A work record can contain a maximum of 255 characters. You can add a maximum of 20 work records. If the number of work records exceeds 20, you need to delete the old work records before adding new ones.
  • Work records are visible to all users and editable by all users.

Procedure

Table 1 Work record operations

Operation

Procedure

Add a work record.
  1. Click Add.
  2. Add a work record in the displayed text box, and click Save.

Modify a work record.
  1. Locate the work record to be modified.
  2. Click and modify the content of the work record in the text box.
  3. Click Save.

Delete a work record.
  1. Locate the work record to be deleted.
  2. Click .
  3. In the Confirm dialog box displayed, click OK.
