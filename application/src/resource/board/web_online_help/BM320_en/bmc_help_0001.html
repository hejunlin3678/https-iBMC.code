Welcome to the Online Help

The Intelligent Baseboard Management Controller (BMC) is designed to monitor and manage a server. The BMC has the following features:

  • Optimal user experience

    The BMC provides a user-friendly graphical user interface (GUI) to facilitate server operation and maintenance (O&M) and improve user experience.

  • Efficient management and maintenance

    The BMC provides remote server management and monitoring.

  • Secure system access

    The BMC uses a variety of mechanisms to ensure secure access to its management interfaces.

The BMC Help provides information about how to use the BMC WebUI. The units of terabyte (TB), gigabyte (GB), megabyte (MB), kilobyte (KB), or byte (B) are converted in the 1024-decimal mechanism.

The BMC provides a WebUI and a command line interface (CLI) to facilitate server O&M.

BMC Clients

The clients used to log in to the BMC WebUI must meet the requirements listed in Table 1. If the remote console needs to be used, use the browser and Java of the correct version.

To ensure that you can browse the complete BMC WebUI page, use the following screen resolutions:

  • 1280 × 800
  • 1366 × 768
  • 1440 × 900
  • 1600 × 900
  • 1600 × 1200
  • 1680 × 1050
  • 1920 × 1080
  • 1920 × 1200
If the TLS version is set to TLS 1.3 only on the User and Security > Security Mgmt page, the BMC runtime environment does not support the following browser versions:
  • Safari 11.0 to 12.0
  • Microsoft Edge 12 to 18
