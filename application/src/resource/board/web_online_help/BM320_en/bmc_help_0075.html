TPCM

Function Description

On the TPCM page, you can view the firmware measurement information of the current server and configure the server of the external trusted management center.

Editing Configurations

Only the administrator can edit configurations when the TSB is enabled and the system is not locked.

  1. Click Configure.
  2. In the displayed Configure window, edit the configuration.

    • Click the button on the right of Measurement. Enable or disable the Measurement.
      • The default value is enabled.
      • If the trustworthiness measurement function is disabled, malicious tampering with firmware cannot be detected immediately.
      • After the measurement function is enabled, you need to update the firmware base value when the firmware is upgraded.
    • Click the button on the right of Control. Enable or disable the Control.
      • The default value is disabled.
      • If the control function is disabled, the firmware cannot be stopped immediately when it is maliciously tampered with, resulting in system security risks.
      • The Control is editable only when the Measurement is enabled.
      • After the control function is enabled, if the firmware fails to pass the measurement, the startup of the firmware will be blocked.
    • Enter the address in the text box of Trusted Server Address. Edit the trusted server address.
      Address format: IPv4, IPv6, or domain name
      The domain name must meet the following requirements:
      • Contain 0 to 255 characters.
      • Allow digits, letters, hyphens (-), and dots (.) only.
      • Cannot start with a hyphen or dot, or end with a hyphen.
      • Allow a maximum of 63 characters between any two dots.

  3. Click OK.

    The Confirm dialogue box is displayed.

  4. Enter the login passward, and click OK.

Base Value Update

  • Only the administrator can update the base value when the TSB is enabled and the system lock is disabled.
  • Update the base value only when the firmware is upgraded. Otherwise, do not change the base value.
  1. Click Update.

    The Update dialog box is displayed.
    • If the BIOS firmware is not upgraded through BMC, you need to power on the OS first after the BIOS firmware upgrade is completed, then reset BMC to ensure that the displayed actual measurement value reflects the latest firmware measurement. Then update the base value.
    • To update the base value of other firmware, reset the OS after the firmware is upgraded, and ensure that the displayed actual measurement value is the latest firmware measurement information. Then update the base value.

  2. (Optional) Change the base value in the text box.

    The base value consists of:

    • 64 characters.
    • Digits (0-9) and letters (a-f, A-F).

  3. Click Save.

    A confirmation dialog box is displayed.

  4. Enter the login password of the current user and click OK.

    • The BIOS firmware base value is successfully updated. Reset the BMC again to make the base value effective, so that the actual measurement value is consistent with the base value.
    • After the base value of other firmware is updated, reset the OS to make the base value effective, so that the actual measurement value is consistent with the base value.
