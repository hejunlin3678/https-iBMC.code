USB Management

Function Description

The USB Management page allows you to query and configure the BMC direct connect management port.

The USB management function is used with the BMC direct connect management port on the server.

Parameter Description

Table 1 USB Management

Parameter

Description

USB Management

After this function is enabled, you can access the BMC through the USB port on the mounting ear of the server by using:

  • Local PC: You can access the BMC WebUI (IP address 169.254.1.5) by using a browser or access the BMC CLI using SSH from a local PC.
  • Mobile device: You can use the mobile application SmartServer to access the BMC from a mobile device.
NOTE:
  • This parameter is enabled by default.
  • Only Windows 10 PCs, Android phones support this function.

USB Device Status

USB device status.

  • Connected: A device is accessed from the USB port.
  • Not Connected: No device is accessed from the USB port.
