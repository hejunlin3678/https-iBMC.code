Online Users

Function Description

On the Online Users page, you can perform the following operations:

  • View the users who have logged in to the BMC.
  • Forcibly log out online users.

Only an administrator can forcibly log out users.

Parameter Description

Table 1 Online Users

Parameter

Description

No.

Sequence number of a logged-in user.

User Name

Name of the user who has logged in to the BMC or KVM.

Login Method

Mode for a user to log in.

The options are as follows:

  • GUI(SSO): A user has logged in to the BMC WebUI over SSO.
  • GUI: A user has logged in to the BMC WebUI in non-SSO mode.
  • CLI: A user has logged in to the BMC over the command-line interface (CLI).
  • KVM: A user has logged in to the OS over the Remote Virtual Console.
  • Redfish: A user has logged in to the BMC over the Redfish interface.
  • VNC: A user logs in to operating system (OS) over the Virtual Network Console.

IP Address

IP address for connecting to and logging in to the BMC.

Value: IP address or COM

NOTE:

The value COM indicates that the user logs in to the BMC over the serial port.

Login Time

Time when the user logged in to BMC.

Operation

Forcibly logs out a user.

Click Log Out to forcibly log out the user.
