Parameter Description

Table 1 Others

Parameter

Description

BBU
Provides battery backup unit (BBU) information (only for the server that supports BBUs):

  • Maximum and actual number of BBUs

  • Name, ID, firmware version, remaining power, working status, battery model, serial number and manufacturer of each BBU

CPU Board
Provides CPU board information:
  • Maximum and actual number of present CPU boards
  • Name, manufacturer, slot number, type, PCB version, CPLD version, board ID, component UID, and power of each CPU board

Drive Backplanes

Provides drive backplane information:

  • Maximum number of expandable drive backplanes and the number of in-position drive backplanes.
  • Name, location, manufacturer, No., type, PCB version, CPLD version, board ID, component UID, part number, and serial number of each drive backplane

Expander
Provides expander board information (only for the server that supports expander boards):
  • Maximum and actual number of expander boards
  • Name, board ID, component UID, CPLD version, No., description, location, manufacturer, PCB version, part number and serial number

Fan backplane

Provides fan backplane information:

  • Maximum and actual number of fan backplane
  • Name, location, manufacturer, type, PCB version, board ID and component UID.

M.2 Adapter

Provides M.2 adapter information, including name, description, slot, PCB version, board ID, part number, and serial number (only for the server that supports M.2 adapters).

OCP Cards
Provides OCP card information (only for the server that supports OCP cards):
  • Maximum number of OCP cards and number of present OCP cards
  • Description, location, manufacturer, slot number, vendor ID, device ID, sub-vendor ID, sub-device ID, and connected resource of each OCP card

PCIe Cards
Provides PCIe card information:
  • Maximum number of PCIe cards and number of present PCIe cards
  • Description, location, manufacturer, slot number, vendor ID, device ID, part number, sub-vendor ID, sub-device ID, and connected resource of each PCIe card

Riser Cards
Provides riser card information:
  • Maximum and actual number of riser cards
  • Name, manufacturer, slot, type, PCB version, board ID, component UID, part number, and serial number of each riser card

Security Module
Provides security module information:
  • Maximum and actual number of security modules
  • Specification type, specification version, manufacturer, manufacturing version, and self-test status of each security module
