Product Info

Parameter Description

Table 1 Product Info

Parameter

Description

Product Info

Product Name

Server model.

Manufacturer

Manufacturer of the server.

Asset Label

Asset tag of the server.

Value: a string of 0 to 48 bytes, allowing digits, letters, and special characters.

NOTE:

Common users cannot set this parameter. Only the administrator, operator, and custom user with the Basic Mgmt rights can set the asset tag of a server.

Serial Number

Serial number of the server.

Part Number

Part number of the server.

Location

Location of the server.

Value: a string of 0 to 64 characters, consisting of digits, letters, and special characters.

BIOS Version

Basic input/output system (BIOS) version.

BMC Firmware Version

BMC firmware version.

BMC Primary U-Boot Version

Version of the primary image of the Universal Boot Loader (U-Boot).

BMC Secondary U-Boot Version

Version of the secondary image of the U-Boot.

System Software

NOTE:
  • The system software information can be displayed on the System Info page only after BMA 2.0 has been installed and restarted on the server OS.
  • If BMA 2.0 has not been installed on the server, obtain the latest BMA documentation and software package and install BMA 2.0.

Computer Name

Computer name defined by the server OS.

OS Version

OS version.

Domain/Workgroup

Domain name or workgroup on the OS of the server.

BMA Service

BMA version.

BMA Running Status

BMA running status.

BMA Driver

BMA driver version.

OS Kernel Version

Kernel version if the Linux OS is used.

Warranty Info

Product Name

Server model.

Serial Number

Serial number of the server.

Production Date

Production date of the server.

UUID

UUID of the server.

Type

Type of the unit of the object for which the service is provided.

Service Start Time

Start date of the service.

Default value: server production date + 100

Lifespan (months)

Time period (in months) for which the service lasts.

Value range: 1 to 255

This warranty info is displayed on the WebUI only when the lifespan set.
