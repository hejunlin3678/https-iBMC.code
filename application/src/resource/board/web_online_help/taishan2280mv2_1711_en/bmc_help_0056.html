Parameter Description

Table 1 Product Info

Parameter

Description

Product Info

Product Name

Server model.

Manufacturer

Manufacturer of the server.

Asset Label

Asset tag of the server.

Value: a string of 0 to 48 bytes, allowing digits, letters, and special characters.

NOTE:

Common users cannot set this parameter. Only the administrator, operator, and custom user with the Basic Mgmt rights can set the asset tag of a server.

Serial Number

Serial number of the server.

Part Number

Part number of the server.

Location

Location of the server.

Value: a string of 0 to 64 characters, consisting of digits, letters, and special characters.

BMC firmware version.

Version of the primary image of the Universal Boot Loader (U-Boot).

BMC Secondary U-Boot Version

Version of the secondary image of the U-Boot.

PCB Version

Printed circuit board (PCB) version.

Board ID

Board ID.

Mainboard Manufacturer

Mainboard Model

Mainboard model.

Mainboard serial number.

Part Number

Part number of the mainboard.

System Software

NOTE:
  • The system software information can be displayed on the System Info page only after BMA 2.0 has been installed and restarted on the server OS.
