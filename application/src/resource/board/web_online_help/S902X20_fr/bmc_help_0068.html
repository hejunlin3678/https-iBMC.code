Gestion Smart Provisioning

Présentation de la fonction

Le Smart Provisioning est un outil utilisé pour le déploiement intelligent des serveurs.

Sur la page Gestion Smart Provisioning, vous pouvez activer ou désactiver la fonction Collecter les informations sur le serveur.

Description du paramètre

Tableau 1 Gestion Smart Provisioning

Paramètre

Description

Collecter les informations sur le serveur

Définition de la fonction de collecte des informations sur le serveur.

Si cette fonction est activée, le système en bande accède au Smart Provisioning pour recueillir des informations sur le serveur lorsque le système en bande démarre pour la première fois à la mise sous tension des PSU. Ensuite, le serveur démarre en fonction des réglages de Support de démarrage préféré et Priorité de démarrage dans Paramètres du BIOS.

Il est désactivé par défaut.
