Ventilation & dissipation

Présentation de la fonction

Sur la page Ventilateur et dissipation thermique, vous pouvez effectuer les opérations suivantes :

  • Consulter les données historiques de la température d'admission d'air du serveur.
  • Consulter et configurer les paramètres de réglage du ventilateur.

    Si le mode de réglage de la vitesse du ventilateur est Manual, tous les réglages sur la page Refroidissement intelligent ne prendra effet que lorsque le mode de réglage de la vitesse du ventilateur passera en mode Auto.

  • Consulter les informations sur les ventilateurs installés sur le serveur.

Description du paramètre

Tableau 1 Température d'admission d'air

Paramètre

Description

Température d'admission

Données de température d'admission d'air échantillonnées toutes les 10 minutes au cours de la dernière semaine.

Valeur actuelle

La dernière température détectée par le capteur d'admission d'air.

Effacer

Cliquez sur Effacer pour effacer toutes les données.

Actualiser le rapport

Cliquez sur Actualiser le rapport pour mettre à jour l'état actuel des statistiques.
Tableau 2 Informations sur le ventilateur

Paramètre

Description

Informations de base

Informations de base sur les modules de ventilation du serveur, notamment le numéro d'emplacement, le nom, le modèle, la vitesse, numéro de pièce et le rapport de vitesse de chaque module de ventilation.

Rafraîchissement en temps réel

Active ou désactive le rafraîchissement en temps réel des informations sur les ventilateurs.

Une fois activé, la vitesse du ventilateur et le rapport de vitesse sont automatiquement rafraîchis toutes les secondes.

Il est désactivé par défaut.
Tableau 3 Paramètres de refroidissement intelligents

Parameter

Description

Mode économie d'énergie

C'est le mode par défaut pour le système de refroidissement par air.

Ajuste les ventilateurs à la charge du système pour une meilleure dissipation de la chaleur et des économies d'énergie.

Mode à faible bruit

Minimise la vitesse du ventilateur pour dissiper la chaleur silencieusement.

Mode haute performance

Maximise la vitesse du ventilateur pour optimiser le refroidissement des composants et le fonctionnement du système.

Mode personnalisé
Fournit des interfaces permettant aux utilisateurs de personnaliser les paramètres suivants. Des plages de valeurs optionnelles seront données à titre de référence en fonction de la charge du serveur et de la situation de dissipation de la chaleur.
  • Température cible du processeur
  • Vitesse du ventilateur
REMARQUE:
  • La vitesse du ventilateur correspondant à une plage de température plus élevée doit être supérieure à la vitesse du ventilateur correspondant à une plage de température inférieure.
  • Les paramètres disponibles dans le mode personnalisé varient en fonction du modèle de serveur.
  • Lors du remplacement du processeur, si la température de fonctionnement la plus élevée du nouveau processeur est inférieure à la valeur actuelle de Température cible du processeur, l'BMC modifie automatiquement la valeur de Température cible du processeur à la température maximale autorisée par le nouveau processeur.

Réglage du mode de refroidissement intelligent

Par exemple, pour définir le mode de refroidissement intelligent sur Mode personnalisé, suivez les étapes suivantes :

Si le mode personnalisé n'est pas configuré correctement, une dissipation de chaleur insuffisante peut être provoquée.

  1. Cliquez sur Paramètres.
  2. Sélectionnez Mode personnalisé..
  3. Définir Température cible du processeur en fonction de la plage de valeurs affichée.
  4. Régler les vitesses de rotation des ventilateurs en fonction des plages de température d'admission.
  5. Cliquez sur Enregistrer.

    Si « Opération réussie » est affiché, le mode de refroidissement intelligent est défini avec succès.
