Language Management

Function Description

On the Language Management page, you can enable and disable the languages supported by the BMC.

  • Only the administrator and the user authorized for common settings can enable or disable languages.
  • Currently, the BMC supports only Chinese, English, Japanese, Russian and French, Chinese and English are enabled by default. The Japanese, Russian, and French language options can be disabled as required.

Parameter Description

Table 1 Language management

Parameter

Description

No.

Sequence number of a language.

Language Code

Code of a language. For example, en stands for English, zh for Chinese, ja for Japanese, ru for Russian, and fr for French.

Language Name

Language corresponding to the code.

Operation

Enables or disables the specified language.

  • : The language is enabled.
  • : The language is disabled.

Querying the Languages Enabled

  1. Choose BMC Settings from the main menu.
  2. Select Language Management.

    The Language Management page is displayed. You can view the languages enabled.
