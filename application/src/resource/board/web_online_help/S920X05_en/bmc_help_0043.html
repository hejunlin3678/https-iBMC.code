License Management

Function Description

On the License Management page, you can use advanced BMC features through license authorization.

Users can use advanced BMC features within the license validity period. Without the license, only the default features of the standby BMC edition can be used.

The BMC advanced edition provides the following features:

  • Deploy OS through the Redfish interface.
  • Enable Kunpeng acceleration engines, including the hardware Security Engine (SEC), High Performance RSA Engine (HPRE), RAID DIF Engine (RDE), and ZIP engine.
  • Enable system lock mode.

Parameter Description

Table 1 License management

Parameter

Description

Device ESN

Equipment serial number (ESN) generated from the mainboard SN and used for applying for a license.

Install

Installs a license.

NOTE:
  • The license is in the .xml format. The size cannot exceed 1 MB.
  • Do not install a license that has been invalidated. Otherwise, an error message will be displayed.

Delete

Deletes a license.

Export

Exports an installed license.

You can export a license and back up it.

Revoke

Revokes a license.

When a license expires and enters the grace period, you can obtain the revoke code from this page. For example, before replacing a component, you need to click the Revoke button to obtain a revoke code. You can use the revoke code to apply for a new license, and install the new license to the new component.

NOTE:

Do not install a license that has been invalidated.

License Status

The license status can be any of the following:

  • Normal: A commercial license has been installed, the license has not expired, and all authorized features are in normal state.
  • Commissioning: A commissioning license has been installed, the license has not expired, and all authorized features are in normal state.
  • Grace Period: A commercial or commissioning license has been installed, the license has expired and in the grace period, all authorized features will not be used after the grace period expires.
  • Default: A commercial or commissioning license has been installed, and the license grace period has expired.

Revoke Code

A code generated after the revoke button is clicked for an installed license. You can use this code to apply for a new license.

License Info

The license information includes the following:

  • License SN
  • License Type
  • The following two types of license are available:
    • Commercial: A formal license issued for a customer based on the contract. All authorized features have the same validity period. The validity period can be Permanent or a specific time. When the license expires, it enters the grace period, which is 60 days.
    • Trial: A temporary license used for trial use of new features, system commissioning, or brand exhibition. The validity period varies depending on the actual situation. When the license expires, it enters the grace period, which is 60 days.
  • Expiration: End date of the license. It can be Permanent or a specific time.
  • License Level
    • Standard edition (default): License for features provided by the standard BMC edition. Users do not need to buy the license.
    • Advanced edition: License for features provided by the advanced BMC edition. Users can use these features only after buying a license.
  • License Grace Period: Number of days for which the BMC can still be used after the license has expired. It has a fixed value of 60.

Advanced Features

Lists information about the license-controlled features.

The information includes the No., name, feature status, current status, and expiry date of each feature.
