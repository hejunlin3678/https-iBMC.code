Parameter Description

Table 1 Others

Parameter

Description

Drive Backplanes

Provides drive backplane information:

  • Maximum number of expandable drive backplanes and the number of in-position drive backplanes.
  • Name, location, manufacturer, No., type, PCB version, CPLD version, board ID, part number, and serial number of each drive backplane

Riser Cards
Provides riser card information:
  • Maximum and actual number of riser cards
  • Name, manufacturer, slot, type, PCB version, board ID, part number, and serial number of each riser card

RAID Controller Card
Provides RAID controller card information:
  • Maximum and actual number of RAID controller cards
  • Name, location, manufacturer, No., type, PCB version, CPLD version, board ID, associated resources, part number, and serial number of the RAID controller card.

CIC Card

Provides CIC card information:

  • Maximum and actual number of CIC cards
  • Name, manufacturer, PCB version, board ID, description, serial number, and part number of each CIC card

PCIe Cards
Provides PCIe card information:
  • Maximum number of PCIe cards and number of present PCIe cards
  • Description, location, manufacturer, slot number, vendor ID, device ID, part number, sub-vendor ID, sub-device ID, and connected resource of each PCIe card

I/O Board
Provides I/O board information (only for the server that supports I/O boards):
  • Maximum and actual number of I/O boards
  • Name, location, manufacturer, type, firmware version, PCB version, CPLD version, board ID, power, part number, and serial number of each I/O board

BBU
Provides battery backup unit (BBU) information (only for the server that supports BBUs):

  • Maximum and actual number of BBUs

  • Name, ID, firmware version, remaining power, working status, battery model, serial number, manufacturer, and M.2 configuration status of each BBU

Security Module
Provides security module information:
  • Maximum and actual number of security modules
  • Specification type, specification version, manufacturer, manufacturing version, and self-test status of each security module
