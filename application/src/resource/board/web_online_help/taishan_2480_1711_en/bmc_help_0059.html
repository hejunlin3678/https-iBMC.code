  • Complete network information can be displayed on the Network Adapters page only after BMA 2.0 has started.

Ethernet NICs

Displays information about LOMs, and PCIe NICs, which includes the NIC name, manufacturer, model, chip model, chip vendor, PCB version, board ID, connected resource (CPU, PCH or PCIe switch), bus information, and PCIe slot information.

  • Click a NIC name to view detailed port information, which includes the port number, port status, port type, media type, speed, and support for auto-negotiation and full duplex.
    NOTE:

    The port status in the Port Properties area can be any of the following:

    • --: The BMA is not installed on the server, and the physical connection status cannot be obtained.
    • Connect: The BMA is not installed on the server, and the physical link is connected.
    • Disconnect: The BMA is not installed on the server, and the physical link is not connected.
    • NoLink: The BMA has been installed on the server, the port is not connected, but the port status is Up.
    • LinkUp: The BMA has been installed on the server, the port is connected, and the port status is Up.
    • LinkDown: The BMA has been installed on the server, and the port status is Down.
  • Click of a NIC to view its network attributes. The network attributes include the name, firmware version, driver name and version, bus information, MAC address, permanent MAC address, IPv4 address (address/subnet mask/gateway), IPv6 address (address/prefix length/gateway), and VLAN information (VLAN ID, VLAN status, and VLAN priority settings) of each network port.
  • Click at the lower part of the port attribute to display the connection view. The connection view provides information about the switch name, switch connection ID, ID of the port connected to the switch, and the switch port VLAN ID.
  • Click at the lower part of the port attribute to display the DCB information and Ethernet statistics of the NIC.
  • If an optical module is connected to the network port, you can click at the lower part of the port attribute to display the optical module information and alarm threshold information. The optical module information includes the manufacturer, SN, component name, device type, device connector type, Rx_LOS state, Tx_Fault state, wavelength, and device identifier, and current temperature, transmit bias current, transmit power, and receive power.
  • If an electrical cable is connected to the network port, you can click at the lower part of the port attribute to display the cable information. The cable information includes the manufacturer, SN, component name, device type, and device connector type.
NOTE:

If the firmware version of a NIC does not support a network port, the network attribute of the port is empty. For example, a NIC has two network ports: port 1 and port 2. If the firmware of the NIC does not support port 2, the network attribute of port 2 is empty.

FC Adapter
Displays FC adapter information, which includes the adapter name, manufacturer, model, chip model, chip manufacturer, part number and serial number.
  • Click an FC card name to view detailed port information, which includes the port number, FC ID, port type, and status.
  • Click of an FC adapter to view its network attributes, which include the speed, world wide port name (WWPN), world wide node name (WWNN), firmware version, and driver name and version.
  • If an optical module is connected to the network port, you can click at the lower part of the port attribute to display the optical module information. The optical module information includes the manufacturer, SN, manufacturing date, component name, module type, transmit mode, transceiver type, speed, device type, device connector type, transmission distance, RxLosState, TxFaultState, wavelength, device identifier, current temperature(°C), current voltage(V), current transmit bias current(mA), current tx power(mW), current rx power(mW), and major alarm threshold (Upper|Lower).
  • For the FC cards that use the Hi1822 processors and support MCTP, the following information can be displayed:
    • Working Rate
    • Work Mode
    • Optical Module Status
    • Peer Device Credit
    • Local Device Credit
    • Transmit Rate
    • Receive Rate
    • Rate Negotiation Stage

Team

Displays aggregated network port information, which includes the port name, status, working mode, IPv4 and IPv6 information (address/subnet mask/gateway), MAC address, and VLAN information (VLAN ID and whether VLAN and VLAN priority are enabled).

Click a port name to view detailed port information, which includes the NIC name, Eth name, port number, MAC address, and status.

Bridge

Displays bridge port information, which includes the port name, status, IPv4 and IPv6 information (address/subnet mask/gateway), MAC address, and VLAN information (VLAN ID and whether VLAN and VLAN priority are enabled).

  • Click a bridge network port name to view detailed port information, which includes the port name, number, status, type, and media type.
  • Click of a bridge port to view detailed information.
