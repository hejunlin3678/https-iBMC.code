Performance monitoring

Function Description

The Performance Monitoring page provides the following information:

  • CPU usage of the last one hour
  • Memory usage of the last one hour
  • Capacity and usage of drive partitions
  • Bandwidth usage of all network ports

Parameter Description

Table 1 Performance monitoring

Parameter

Description

CPU Usage

Percentage of CPU resources used by applications in running.

NOTE:
  • To view CPU usage, install BMA 2.0 on the OS and start it.
  • If BMA 2.0 has not been installed on the server, obtain the latest BMA documentation and software package and install BMA 2.0.

Memory Usage

Percentage of memory resources used by applications in running.

NOTE:
  • To view memory usage, install BMA 2.0 on the OS and start it.
  • If BMA 2.0 has not been installed on the server, obtain the latest BMA documentation and software package and install BMA 2.0.

Network Port Bandwidth Usage

Percentage of the bandwidth used by all the ports of the server NICs to the total bandwidth.

NOTE:
  • To view network port bandwidth usage, install BMA 2.0 on the OS and start it.
  • If BMA 2.0 has not been installed on the server, obtain the latest BMA documentation and software package and install BMA 2.0.

Drive Usage
The Drive Usage area provides the following information:
  • Percentage of the used partition space to the total partition space
  • Drive partition information
  • Total capacity and usage capacity of each partition
NOTE:
  • To view drive usage, install BMA 2.0 on the OS and start it.
  • If BMA 2.0 has not been installed on the server, obtain the latest BMA documentation and software package and install BMA 2.0.

Current Value

Current CPU or memory usage of the server.

Threshold

Threshold of the CPU, memory, disk, or network port bandwidth usage. If the actual usage exceeds the threshold, the BMC reports an event.

Value range: 0 to 100

Click to set the threshold.

Click to refresh the usage curve.

Click to clear the bandwidth usage statistics of network ports.

Setting the Threshold

  1. In the area, click .

    A dialog box is displayed.

  2. Enter the threshold value based on the value range displayed.
  3. Click to save the settings.
