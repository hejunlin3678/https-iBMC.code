"Setting the Trust Certificate Timed Out" Displayed When the HTML5 Integrated Remote Console Is Opened

Symptom

Symptom

Possible Causes

When a user attempts to open the HTML5 Integrated Remote Console, the system displays "Failed to open the KVM because setting the trust certificate timed out."

Before a connection is set up between the KVM client (HTML5 Integrated Remote Console) and the server, the SSL certificate must be verified. If the verification fails, the connection cannot be set up.

Solution

  1. On the BMC WebUI, choose Services > Web Services. In the Server Certificate Information area, check whether the server certificate has expired.

    • If yes, go to 2.
    • If no, go to 3.

  2. Generate a new certificate and replace the expired certificate.
  3. Restart the BMC.
  4. Check whether the HTML5 Integrated Remote Console can be opened.

    • If yes, no further action is required.
    • If no, go to 5.

  5. Contact technical support.
