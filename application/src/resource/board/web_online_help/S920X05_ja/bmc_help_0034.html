仮想メディア

機能説明

VMM（Virtual Media Manager）サービスを使用することで、仮想CDまたはDVDドライブまたは仮想フロッピードライブをサーバーにマウントできます。

[仮想メディア]画面で、VMM設定を照会・設定できます。

パラメータの説明

表 1 仮想メディア

パラメータ

説明

VMM

VMMサービスの状態。

デフォルトで有効になっています。

ポート

VMMサービスのポート番号。

値は以下のとおりです。1～65535

デフォルト値: 8208

NOTE:

セキュリティ上の理由から、ブラウザはウェブ閲覧に使用しないいくつかのポートを無効にしています。これらのポートが使用されると、HTML5統合リモートコンソールの仮想メディアが利用できなくなります。

最大セッション数

VMMに同時にアクセスできるユーザーの最大数。このパラメータは1に指定されています。

アクティブセッション数

現在VMMに同時にアクセスしているユーザーの数。
