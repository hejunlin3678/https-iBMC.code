リモートバーチャルコンソールでマウスとキーボードを使用できない

故障現象

症状説明

考えられる原因

マウスとキーボードはリモート仮想コンソールでは動作しません。

サーバーにLSISAS3108 RAIDコントローラカードが設定されており、仮想キーボードとマウスの持続接続の設定が無効になっています。

解決策

  1. サーバーにLSISAS3108 RAIDコントローラカードが部品情報画面に設定されているかを確認します。

    • はいの場合、2に移動します。
    • いいえの場合、4に移動します。

  2. 仮想コンソール画面で、仮想キーボードとマウスの持続接続の設定が有効になっているかを確認します。

    • はいの場合、4に移動します。
    • いいえの場合、3に移動します。

  3. 仮想キーボードとマウスの持続接続の設定を有効にしてサーバーを再起動します。サーバーの再起動後、問題が解決しているかを確認してください。

    • はいの場合、これ以上の操作は必要ありません。
    • いいえの場合、4に移動します。

  4. 技術サポートのエンジニアにご連絡ください。
