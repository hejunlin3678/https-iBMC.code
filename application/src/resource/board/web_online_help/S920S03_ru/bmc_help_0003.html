Обзор продукта

Интеллектуальный контроллер управления материнской платой (BMC) предоставляет разнообразные функции и возможности управления.

  • Различные интерфейсы управления для системной интеграции

    BMC поддерживает следующие интерфейсы:

    • Интерфейс управления центром обработки данных v1.5 (DCMI 1.5)
    • Интеллектуальный интерфейс управления платформой (IPMI) 1.5 и 2.0
    • Интерфейс командной строки (CLI)
    • Интерфейс Redfish
    • Интерфейс HTTPS (Hypertext Transfer Protocol Secure, Защищенный протокол передачи гипертекста)
    • Интерфейс SNMP (Simple Network Management Protocol, Простой протокол сетевого управления)
  • Мониторинг и диагностика неисправностей

    BMC обнаруживает скрытые угрозы и обеспечивает стабильную бесперебойную работу системы в режиме 24/7, предоставляя следующие функции:

    • Создание скриншотов и видео в случае отказа системы помогают определить причину сбоя системы.
    • Создание мгновенных снимков экрана и запись видео упрощают плановое профилактическое техобслуживание, запись информации и аудит.
    • BMC поддерживает отправку отчетов об аварийных сигналах в систему управления сетью (NMS, Network Management System) верхнего уровня посредством функций syslog, trap и электронной почты.
  • Управление безопасностью
    • Для повышения безопасности системы BMC поддерживает функцию зеркалирования образов. При выходе из строя работающего программного обеспечения система имеет возможность запуска с помощью резервного образа.
    • Разнообразные интерфейсы управления безопасностью обеспечивают надежный контроль доступа пользователей к системе.
    • BMC поддерживает импорт и замену сертификатов разных типов для обеспечения безопасности передачи данных.
  • Интерфейсы техобслуживания системы
    • BMC поддерживает подключение клавиатуры, монитора и мыши (KVM, Keyboard, Video, Mouse), а также виртуальных носителей для реализации удаленного обслуживания.
    • BMC поддерживает функцию внеполосного мониторинга и настройки RAID, которая повышает эффективность конфигурации и управления RAID.
    • Функция Smart Provisioning обеспечивает установку ОС , настройку RAID и обновление без использования DVD-дисков, упрощая установку и настройку сервера.
  • Различные сетевые протоколы
    • BMC поддерживает протокол сетевого времени (NTP, Network Time Protocol) для настройки и синхронизации времени сервера.
    • BMC поддерживает управление доменами и службы каталогов для упрощения процесса управления сетью.
  • Интеллектуальное управление питанием
    • С помощью технологии ограничения энергопотребления повышается плотность развертывания.
    • BMC имеет функцию динамического энергосбережения для снижения операционных расходов (OPEX).
  • Управление лицензиями

    Функция управления лицензиями обеспечивает авторизованным пользователям доступ к дополнительным функциям.

    BMC предоставляет следующие дополнительные функциональные возможности:

    BMC предоставляет следующие расширенные функциональные возможности:
    • Развертывание ОС посредством интерфейса Redfish.
    • Сбор первоначальных данных для интеллектуальной диагностики посредством интерфейса Redfish.
    • Включение режим блокировки системы.
