Информация об устройстве

Краткое описание сервера, включая:

  • Серийный номер продукта: серийный номер сервера.
  • Имя хоста: название хоста BMC.
  • Имя домена: указывает имя домена.
  • Адрес IPv4 BMC или Адрес IPv6 BMC: IP-адрес, используемый для входа в BMC.
  • Версия встроенного ПО BIOS: версия встроенного программного обеспечения базовой системы ввода-вывода.
  • MAC-адрес: Физический адрес сетевого порта BMC.
  • GUID: глобально уникальный идентификатор сервера (Графический интерфейс пользователя (GUI)D, Globally Unique IDentifier).

Можно нажать Подробнее, чтобы перейти к Система > Данные о системе > Данные о продукте.

Мониторинг системы

Предоставляет быстрые доступы к следующим объектам мониторинга:

  • Температура поступающего воздуха: нажмите для перехода на страницу Система > Вентиляторы и теплоотвод.
  • Энергия: нажмите для перехода на страницу Система > Энергия > Ограничение энергопотребления.
ПРИМЕЧАНИЕ:
  • Если текущая температура воздуха на входе меньше порогового значения уровня-1, значение температуры сервера соответствует допустимому диапазону.
  • Если текущая температура воздуха на входе больше или равна пороговому значению уровня-1 и меньше порогового значения уровня-2, значение температуры сервера превышает допустимый диапазон, поэтому необходимо принять корректирующие меры.
  • Если текущая температура воздуха на входе больше или равна пороговому значению уровня-2, значение температуры превышает значение, установленное для генерирования аварийного сигнала критического уровня. Корректирующие меры должны быть приняты немедленно.

Состояние работоспособности

Предоставляет быстрые доступы к следующим системным ресурсам:

  • ЦПУ: нажмите для перехода на страницу Система > Информация > Процессоры.
  • Память: нажмите для перехода на страницу Система > Информация > Память.
  • Хранилище: нажмите для перехода на страницу Система > Управление хранилищем.
  • Сетевые адаптеры: нажмите для перехода на страницу Система > Данные о системе > Сетевые адаптеры.
  • Энергия: нажмите для перехода на страницу Система > Энергия > Энергопотребление.
  • Вентиляторы: нажмите для перехода на страницу Система > Вентиляторы и теплоотвод.

Виртуальная консоль

Запуск встроенной удаленной консоли Java или HTML5.

Щелкните кнопкой мыши Удаленная консоль и в появившемся диалоговом окне выберите встроенную удаленную консоль Java (приватную или совместную) или HTML5 (приватную или совместную).

Нажмите Начать, чтобы перейти на страницу Виртуальная консоль.

Более подробная информация о виртуальной консоли приводится в разделах:
  • Описание функций
  • Встроенная удаленная консоль HTML5
  • Встроенная удаленная консоль Java
  • Устранение неполадок удаленной виртуальной консоли

Быстрый доступ

Панель предоставляет следующие кнопки быстрого доступа:

  • Локальные пользователи: быстрый переход на страницу Пользователи и безопасность > Локальные пользователи.
  • Сеть: быстрый переход на страницу Настройки BMC > Настройки сети.
  • Энергопотребление: быстрый доступ к Система > Энергия > Управление энергопотреблением.
  • Обновление встроенного ПО: быстрый переход на страницу Настройки BMC > Обновление встроенного ПО.
  • Сбор данных одним нажатием: сбор информации по техническому обслуживанию. Более подробная информация о сборе данных приводится в разделе Сбор данных одним нажатием.
  • Восстановить заводские настройки: восстановление заводских настроек.

    Все пользовательские настройки будут сброшены. Восстановление заводских настроек затронет помимо прочего:

    • Состояние подключения последовательного порта
    • Настройки ограничения энергопотребления
    • После восстановления заводских настроек будут удалены сертификаты LDAP и SSL, выгруженные пользователями.
    • Имена пользователей, пароли, сроки действия аккаунтов, настройки групп пользователей и настройки блокировки пользователей
    • Режим назначения IP-адресов, IP-адреса, маски подсети и шлюзы по умолчанию
    • Настройки SNMP
    • Настройки SNMP trap и SNMP для передачи аварийной информации

Информация о последнем входе в систему

Отображает информацию о последнем входе текущего пользователя в первые 10 секунд после его входа в систему.

В информацию включены следующие сведения:

  • Имя пользователя

  • IP-адрес

  • Время последнего входа
