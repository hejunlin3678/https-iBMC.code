Описание функций

Удаленная консоль предназначена для получения доступа к операционной системе сервера. BMC поддерживает встроенную удаленную консоль HTML5 и встроенную удаленную консоль Java.

Встроенная удаленная консоль HTML5

Встроенная удаленная консоль HTML5 предоставляет два режима доступа:

  • В приватном режиме доступ к серверу и право выполнения операций на нем через BMC предоставляется только одному локальному пользователю или пользователю VNC.
  • В совместном режиме доступ и право выполнения операций на сервере через BMC предоставляется одновременно двум локальным пользователям или пяти пользователям VNC. Каждый пользователь может посмотреть, какие операции выполняют другие пользователи, работающие одновременно с ним.

Встроенная удаленная консоль HTML5 позволяет выполнять такие операции, как:

  • Настройка экрана в соответствии с предпочтениями пользователя с помощью кнопок закрепления окон, кнопок масштабирования изображения, кнопок мыши и регулятора Четкость изображения.
  • Настройка загрузочного устройства с помощью комбинаций ключевых клавиш и клавиш раскладки клавиатуры.
  • Регулирование работы ОС сервера с помощью кнопок управления энергопотреблением и запись операций, выполняемых в ОС, с помощью кнопки видеозаписи.
  • Подключение файла образа или локальной папки с помощью кнопок DVD и FDD.

Встроенная удаленная консоль Java

Встроенная удаленная консоль Java предоставляет два режима доступа:

  • В приватном режиме доступ к серверу и право выполнения операций на нем через BMC предоставляется только одному локальному пользователю или пользователю VNC.
  • В совместном режиме доступ и право выполнения операций на сервере через BMC предоставляется одновременно двум локальным пользователям или пяти пользователям VNC. Каждый пользователь может посмотреть, какие операции выполняют другие пользователи, работающие одновременно с ним.

Встроенная удаленная консоль позволяет выполнять такие операции, как:

  • Настройка экрана в соответствии с предпочтениями пользователя с помощью кнопок закрепления окон, кнопок масштабирования изображения, кнопок мыши и регулятора Четкость изображения.
  • Настройка загрузочного устройства с помощью комбинаций ключевых клавиш, клавиш со стрелками и клавиш раскладки клавиатуры.
  • Регулирование работы ОС сервера с помощью кнопок управления энергопотреблением и запись операций, выполняемых в ОС, с помощью кнопки видеозаписи.
  • Подключение физических дисков DVD и FDD, файла образа или локальной папки.
  • Создание DVD-диска или файла образа программного обеспечения с помощью кнопки создания образа.

Открытие удаленной виртуальной консоли

При вводе пароля ОС или BIOS на удаленной виртуальной консоли:

  • Используйте имеющуюся клавиатуру, если ее настройки соответствуют настройкам клавиатуры ОС.
  • Используйте клавиатуру ОС, если настройки используемой клавиатуры не соответствуют настройкам клавиатуры ОС.

Пользователь может проигнорировать предупреждение системы безопасности или выполнить любое из следующих действий, чтобы его скрыть:

  • Импортировать доверенный сертификат и корневой сертификат в BMC.
  • Если доверенного сертификата нет в наличии, добавьте BMC в список надежный сайтов на панели управления Java. Однако данная операция представляет угрозу безопасности.
  • На Главной странице WebUI BMC выберите в выпадающем списке встроенную удаленную консоль: Java или HTML5.

    В совместном режиме удаленной консоли доступ и право выполнения операций на сервере предоставляется одновременно двум пользователям. Каждый пользователь может посмотреть, какие операции выполняет другой пользователь, работающий одновременно с ним.

    В приватном режиме удаленной консоли доступ и право выполнения операций на сервере предоставляется только одному пользователю. В данном режиме функция создания скриншотов вручную будет недоступна.

  • Откройте браузер и введите в адресную строку:
    • Любое из следующих значений:
      • Для входа в встроенную удаленную консоль HTML5 введите:
        • https://IP address/remoteconsole?openWay=html5 или https://IP address/remoteconsole?openway=html5
        • https://IP address/remote_access.asp?authParam=key&lp=lang&openWay=html5 или https://IP address/remote_access.asp?authParam=key&lp=lang&openway=html5
      • Для входа в встроенную удаленную консоль Java введите:
        • https://IP address/remoteconsole, https://IP address/remoteconsole?openWay=jre или https://IP address/remoteconsole?openway=jre
        • https://IP address/remote_access.asp?authParam=key&lp=lang&openWay=jre или https://IP address/remote_access.asp?authParam=key&lp=lang&openway=jre
      • key можно настроить с помощью интерфейса Redfish для прямого подключения к KVM.
      • lp указывает на тип языка, используемого на консоли.
      • Параметр openWay может быть только openway или openWay. Остальные значения приведут к переключению на консоль Java.
    • https://IP address/kvmvmm.asp
    • https://IP address/login.html?redirect_type=1

    IP address — это IP-адрес BMC.
