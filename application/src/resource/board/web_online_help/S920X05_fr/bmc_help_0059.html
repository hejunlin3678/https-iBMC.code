Cartes réseau

Affiche des informations sur les cartes mère (LOM) et les cartes réseau (NIC) PCIe qui incluent le nom de la carte réseau (NIC), le fabricant, le modèle, le modèle de la puce, le fournisseur de la puce, la version du circuit imprimé, l'ID de la carte, la ressource connectée ( processeur, commutateur PCH ou PCIe), les informations de bus et les informations sur les emplacements PCIe.

  • Cliquez sur un nom de carte réseau pour consulter des informations détaillées sur le port, notamment le numéro de port, l'état du port, le type de port, le type de support, la vitesse, la prise en charge de la négociation automatique et du duplex intégral.
    REMARQUE:

    L'état du port dans la zone Propriétés du port peut être l'un des suivants :

    • -- : BMA n'est pas installé sur le serveur, et l'état de la connexion physique ne peut être obtenu.
    • Connexion : BMA n'est pas installé sur le serveur, et le lien physique est établi.
    • Se déconnecter : BMA n'est pas installé sur le serveur, et le lien physique n'est pas établi.
    • NoLink : BMA a été installé sur le serveur, le port n'est pas connecté, mais l'état du port est Up.
    • LinkUp : BMA a été installé sur le serveur, le port est connecté, et l'état du port est Up.
    • LinkDown : BMA a été installé sur le serveur, et l'état du port est Down.
  • Cliquez sur d'une carte réseau pour afficher ses attributs réseau. Les attributs réseau incluent le nom, la version du micrologiciel, le nom et la version du pilote, les informations de bus, l'adresse MAC, l'adresse MAC permanente, l'adresse IPv4 (adresse/masque de sous-réseau/passerelle), l'adresse IPv6 (adresse / longueur du préfixe / passerelle) et les informations VLAN (ID VLAN, état VLAN et paramètres de priorité VLAN) pour chaque port réseau.
  • Cliquez sur dans la partie inférieure de Propriétés du port pour afficher la vue de connexion. La vue de connexion fournit des informations sur le nom du commutateur, l'ID de connexion du commutateur, l'ID du port connecté au commutateur et l'ID du port VLAN du commutateur.
  • Cliquez sur dans la partie inférieure de Propriétés du port pour afficher les informations DCB et les statistiques Ethernet de la carte d'interface réseau.
  • Si un module optique est connecté au port réseau, vous pouvez cliquer sur dans la partie inférieure de Propriétés du port pour afficher les informations sur le module optique et le seuil d'alarme. Les informations du module optique incluent le fabricant, le numéro de série, le nom du composant, le type du dispositif, le type du connecteur du dispositif, l'état de perte de réception, l'état de défaut d'émission, la longueur d'ondes et l'identifiant du dispositif, la température actuelle, le courant de transmission actuel, la puissance de transmission actuelle et la puissance de réception actuelle.
  • Si un câble électrique est connecté au port réseau, vous pouvez cliquer sur la partie inférieure de Propriétés du port pour afficher les informations sur le câble. Les informations sur le câble incluent le fabricant, le SN, le nom du composant, le type du dispositif et le type du connecteur du dispositif.
REMARQUE:

Si la version du micrologiciel d'une carte réseau ne prend pas en charge un port réseau, l'attribut réseau du port est vide. Par exemple, une carte réseau a deux ports réseau : le port 1 et le port 2. Si le micrologiciel de la carte réseau ne prend pas en charge le port 2, l'attribut réseau du port 2 est vide.

Adaptateur FC
Affiche les informations de l'adaptateur FC, y compris le nom de l'adaptateur, le fabricant, le modèle, le modèle et le fabricant de la puce, le numéro de pièce et le numéro de série.
  • Cliquez sur le nom d'une carte FC pour afficher des informations détaillées sur le port, notamment le numéro de port, l'ID FC, le type de port et le statut.
  • Cliquez sur un adaptateur FC pour afficher ses attributs réseau, notamment la vitesse, le nom du port mondial (WWPN), le nom du nœud mondial (WWNN), la version du micrologiciel, ainsi que le nom et la version du pilote.
  • Si un module optique est connecté au port réseau, vous pouvez cliquer sur Cliquez sur dans la partie inférieure de Propriétés du port pour afficher les informations sur le module optique. Les informations du module optique incluent le fabricant, le numéro de série, le date de fabrication, le nom du composant, la type de module, le mode de transmission, la type d'émetteur-récepteur, le vitesse, la type de périphérique, le type du connecteur du dispositif, le distance de transmission, l'état de perte de réception, l'état de défaut d'émission, la longueur d'ondes, l'identifiant du dispositif, la température actuelle(℃), la tension actuelle (V), le courant de transmission actuel(mA), la puissance de transmission actuelle(mW), la puissance de réception actuelle(mW) et le seuil d'alarme majeure (supérieur|inférieur).
  • Pour les cartes FC utilisant les processeurs Hi1822 et prenant en charge MCTP, les informations suivantes peuvent être affichées :
    • Taux d'activité
    • Mode de travail
    • État du module optique
    • Crédit d'appareil pair
    • Crédit d'appareil local
    • Taux de transmission
    • Taux de réception
    • Stade de négociation de taux

Équipe

Affiche des informations agrégées sur le port réseau, y compris le nom, l'état, le mode de fonctionnement, les informations IPv4 et IPv6 (adresse/masque de sous-réseau/passerelle), l'adresse MAC et les informations VLAN (ID VLAN et si la priorité VLAN et VLAN sont activées) du port.

Cliquez sur le nom d'un port pour afficher des informations détaillées sur le port, notamment le nom de la carte réseau, le nom Eth, le numéro de port, l'adresse MAC et l'état.

Passerelle

Affiche les informations sur le port de la passerelle, y compris le nom, l'état, les informations IPv4 et IPv6 (adresse/masque de sous-réseau/passerelle), l'adresse MAC et les informations VLAN (ID VLAN et si la priorité VLAN et VLAN sont activés) du port.

  • Cliquez sur le nom d'un port réseau de passerelle pour afficher des informations détaillées sur le port, y compris le nom, le numéro, l'état, le type et le type de support du port.
  • Cliquez sur un port de passerelle pour afficher des informations détaillées.
