Bienvenue à l'aide BMC

Le contrôleur de gestion de la carte mère intelligente (BMC) est désigné pour surveiller et gérer un serveur. Le BMC possède les caractéristiques suivantes :

  • Expérience d'utilisateur optimale

    Le BMC fournit une interface utilisateur graphique conviviale (GUI) pour faciliter l'opération et la maintenance (O&M) du serveur et améliorer l'expérience d'utilisateur.

  • Gestion et maintenance efficaces

    Le BMC fournit la gestion et la surveillance des serveurs à distance.

  • Accès sécurisé au système

    Le BMC utilise divers mécanismes pour assurer un accès sécurisé à ses interfaces de gestion.

Le BMC aide à fournir des informations sur l'utilisation de l'interface utilisateur Web BMC. Les unités de téraoctets (To), gigaoctets (Go), mégaoctets (Mo), kilooctets (Ko) ou octets (o) sont converties dans le mécanisme décimal 1 024.

Le BMC fournit une interface utilisateur Web et une interface de ligne de commande (CLI) pour faciliter l'opération et la maintenance (O&M) du serveur.

Clients BMC

Les clients utilisés pour se connecter à BMC WebUI doivent répondre aux exigences répertoriées dans le Tableau 1.. Si la console distante doit être activée, utilisez le navigateur et Java de la version appropriée.

Pour vous assurer que vous pouvez naviguer sur les pages complètes de l'interface utilisateur Web BMC, utilisez les résolutions d'écran suivantes :

  • 1280 × 800
  • 1366 × 768
  • 1440 × 900
  • 1600 × 900
  • 1600 × 1200
  • 1680 × 1050
  • 1920 × 1080
  • 1920 × 1200
Si la version TLS est réglée sur TLS 1.3 uniquement sur la page Utilisateur et sécurité > Gestion de la sécurité, l'environnement d'exécution BMC ne prend pas en charge les versions de navigateurs suivants :
  • Safari 11.0 à 12.0
  • Microsoft Edge 12 à 18
