Connexion

Présentation de la fonction

La page Connexion permet d'accéder à l'interface utilisateur Web de le BMC.

  • Un maximum de quatre utilisateurs peuvent se connecter simultanément à l'interface utilisateur Web BMC.
  • Par défaut, le délai d'expiration du système est de 5 minutes. Si aucune opération n'est effectuée sur l'interface utilisateur Web dans les 5 minutes, l'utilisateur sera automatiquement déconnecté de l'interface utilisateur Web.
  • Si le nombre de tentatives consécutives de saisie des mots de passe erronés atteint la valeur spécifiée, le système verrouille le compte de l'utilisateur. L'utilisateur ne peut se connecter au système qu'après l'expiration de la durée de verrouillage.
  • Pour des raisons de sécurité, changez le mot de passe initial après la première connexion et changez votre mot de passe périodiquement.
  • Un affichage incorrect peut être provoqué sur l'interface utilisateur Web de le BMC lorsque les ressources ne sont pas obtenues en raison des fluctuations du réseau. Si ce problème survient, actualisez le navigateur et connectez-vous à nouveau à l'interface utilisateur Web de le BMC.

Description du paramètre

Tableau 1 Paramètres sur la page de connexion

Paramètre

Description

Nom d'utilisateur

Saisissez le nom d'utilisateur pour vous connecter à l'interface utilisateur Web BMC.

  • Lorsque Domaine est défini sur BMC local, la longueur maximale du nom d'utilisateur est de 16 caractères.
  • Lorsque Domaine n'est pas défini sur BMC local, la longueur maximale du nom d'utilisateur est de 255 caractères.

Notez les points suivants :

  • Lorsque vous connectez à le BMC en tant qu'utilisateur local, sélectionnez BMC local ou Correspondance automatique pour le domaine.
  • Pour un utilisateur LDAP, le nom d'utilisateur dans les formats suivants est pris en charge :
    • Nom d'utilisateur LDAP (sélectionnez Correspondance automatique ou saisissez le nom de domaine pour le domaine)
    • Nom d'utilisateur LDAP@Nom de domaine (sélectionnez Correspondance automatique ou saisissez le nom de domaine pour le domaine)
  • Pour un utilisateur Kerberos, le nom d'utilisateur dans les formats suivants est pris en charge :
    • Nom d'utilisateur Kerberos (sélectionnez Correspondance automatique ou saisissez le nom de domaine pour le domaine)
    • Nom d'utilisateur Kerberos@Nom de domaine (sélectionnez Correspondance automatique ou saisissez le nom de domaine pour le domaine. Les lettres du nom de domaine doivent être en majuscules)
  • Le BMC prend en charge l'authentification unique (SSO) à l'aide du nom d'utilisateur Kerberos ou du nom d'utilisateur Kerberos@nom de domaine.

Mot de passe

Saisissez le mot de passe pour vous connecter à l'interface utilisateur Web de le BMC.

REMARQUE:

La longueur maximale du mot de passe pour un utilisateur LDAP ou Kerberos qui se connecte à l'interface utilisateur Web BMC est de 255 caractères.

Connexion en tant qu'utilisateur local

  1. Sur la page de connexion de l'interface utilisateur Web BMC, sélectionnez la langue à utiliser.
  2. Saisissez le nom d'utilisateur et le mot de passe.
  3. Sélectionnez BMC local ou Correspondance automatique dans la liste déroulante Domaine.
  4. Cliquez sur Connexion.

    La page Informations s'affiche, indiquant le nom d'utilisateur dans le coin supérieur droit.

Connexion en tant qu'utilisateur LDAP

Avant de vous connecter, assurez-vous que les paramètres suivants répondent aux exigences :

  • Un contrôleur de domaine existe sur le réseau et un domaine utilisateur et des utilisateurs LDAP ont été créés sur le contrôleur de domaine.

    Pour plus d'informations sur la création d'un contrôleur de domaine, d'un domaine utilisateur et des utilisateurs LDAP, reportez-vous aux documents du contrôleur de domaine. Le BMC fournit uniquement la fonction d'accès pour les utilisateurs LDAP.

  • LDAP a été activé dans Utilisateur et Utilisateur et sécurité > LDAP sur l'interface utilisateur Web BMC, et un domaine utilisateur et des utilisateurs LDAP ont été définis.
  1. Sur la page de connexion de l'interface utilisateur Web BMC, sélectionnez la langue à utiliser.
  2. Saisissez le nom d'utilisateur et le mot de passe LDAP.

    • Pour se connecter en tant qu'utilisateur LDAP, le nom d'utilisateur peut être dans l'un des formats suivants :
      • Nom d'utilisateur LDAP

        Si un nom d'utilisateur LDAP est saisi, le Domaine peut être une Correspondance automatique ou un domaine spécifié.

      • Nom d'utilisateur LDAP @ nom de domaine

        Si nom d'utilisateur LDAP @ nom de domaine est saisi, le Domaine peut être une Correspondance automatique ou un domaine spécifié.

    • La longueur maximale du mot de passe permettant à un utilisateur LDAP de se connecter à l'interface utilisateur Web de le BMC est de 255 caractères.

  1. Sélectionnez le domaine utilisateur LDAP dans la liste déroulante Domaine.

    La liste déroulante Domaine contient les options suivantes :

    • BMC local : Sélectionnez cette option pour vous connecter en tant qu'utilisateur local. Le BMC localise automatiquement le compte utilisateur à partir de la liste des utilisateurs locaux.
    • Serveur de domaine configuré : Sélectionnez un serveur de domaine pour vous connecter en tant qu'utilisateur LDAP. Le BMC localise automatiquement le compte utilisateur à partir du serveur de domaine.
    • Correspondance automatique : Si cette option est sélectionnée, le BMC recherche d'abord le compte utilisateur à partir de la liste des utilisateurs locaux. Si aucune correspondance n'est trouvée, le BMC effectue une recherche parmi les serveurs de domaine dans l'ordre indiqué dans la liste déroulante Domaine.

  2. Cliquez sur Connexion.

    La page Informations s'affiche, indiquant le nom d'utilisateur dans le coin supérieur droit.

Connexion en tant qu'utilisateur Kerberos

Environnement d'exécution Kerberos :

  • Le client exécute Windows 10 64 bits.
  • Le serveur Kerberos prend en charge Windows Server 2012 R2 64 bits et Windows Server 2016 64 bits.

Les modes de connexion suivants sont pris en charge :

  • Connectez-vous à l'aide d'un compte de domaine Kerberos
  • Authentification unique（SSO）

Avant de vous connecter, assurez-vous que les conditions suivantes sont remplies :

  • Kerberos a été activé dans Utilisateur et sécurité > Kerberos sur l'interface utilisateur Web BMC, et la fonction et le groupe d'utilisateurs Kerberos ont été configurés.
  • Le groupe et le nom d'utilisateur Kerberos ont été créés sur le serveur Kerberos, et l'utilisateur a été ajouté au groupe d'utilisateurs Kerberos. Cet utilisateur est un utilisateur qui se connecte au système d'exploitation du client.

Pour vous connecter à l'interface utilisateur Web BMC en utilisant un compte de domaine Kerberos, suivez les étapes suivantes :

  1. Sur la page de connexion de l'interface utilisateur Web BMC, sélectionnez la langue à utiliser.
  2. Saisissez le nom d'utilisateur et le mot de passe Kerberos. Pour plus de détails, voir Description du paramètre.
  3. Dans la liste déroulante des domaines, sélectionnez le domaine utilisateur Kerberos, par exemple ADMIN.COM(KRB), ou Correspondance automatique.
  1. Cliquez sur Connexion.

    Si la connexion est réussie, la page Accueil s'affiche.

Pour vous connecter à l'aide de l'authentification unique, suivez les étapes suivantes :

  1. Utilisez le nom d'utilisateur et le mot de passe Kerberos configurés sur le serveur Kerberos pour vous connecter au système d'exploitation client.
  2. Ouvrez le navigateur, saisissez le nom de domaine complet BMC, par exemple https://nom d'hôte.nom de domaine, dans la zone d'adresse et appuyez sur Entrée.

    La page de connexion BMC s'affiche.

  3. Cliquez sur Authentification unique.

    Si la connexion est réussie, la page Accueil s'affiche.
