Alarme et SEL

Présentation de la fonction

Sur la page Alarme et SEL, vous pouvez :

  • Interroger les alarmes actives du serveur.
  • Voir et rechercher les événements système, puis télécharger et effacer tous les événements système.

Description du paramètre

Tableau 1 Alarme et SEL

Paramètre

Description

No.

Numéro séquentiel d'un événement.

Gravité

Gravité d'une alarme.

  • : Une alarme critique peut mettre le serveur hors tension et peut même interrompre des services système. Vous devez prendre des mesures correctives immédiatement.
  • : Une alarme majeure a un impact majeur sur le système. Cela affecte le fonctionnement normal du système ou peut entraîner une interruption de service.
  • : Une alarme mineure a un impact mineur sur le système, mais vous devez prendre des mesures correctives dès que possible pour éviter une alarme plus grave.
  • : Un événement s'est produit pendant le fonctionnement normal du système.

Type d'objet

Type du composant pour lequel l'alarme a été générée.

Statut

Statut de l'événement système.

Valeur :

  • Asserted : indique qu'un événement système est généré.
  • Deasserted : indique qu'un événement système est effacé.

Code d'événement

Identifier de manière unique une alarme.

Créé

Date et heure de génération de l'alarme.

Description d'événement

Informations complémentaires sur l'alarme.

Suggestion de résolution

Suggestions sur la façon d'effacer l'alarme.

Recherche d'événements système

  1. Choisissez Entretien > Alarme et SEL, et sélectionnez l'onglet Événements système.
  2. Cliquez sur Critères de filtrage.

    La page de paramétrage des Critères de filtrage s'affiche.

  3. Définir les Critères de filtrage. Pour plus de détails sur les paramètres, voir Tableau 2.

    Tableau 2 Critères de filtrage.

    Paramètre

    Description

    Inclure les gravités d'alarme

    Gravité de l'événement système.

    Valeur : Tous, Critique, Majeur, Mineur ou Informationnel

    Type d'objet

    Type du composant pour lequel l'événement système a été généré.

    Valeur : varie selon le modèle de serveur.

    Créé

    Date et heure de génération de l'événement système.

    Valeur :

    • TOUT
    • Aujourd'hui
    • Derniers 7 jours
    • Derniers 30 jours
    • Personnaliser
    REMARQUE:

    Si vous sélectionnez Personnaliser, vous devez spécifier les dates de début et de fin.

    Requête d'entrée

    Personnaliser les critères de recherche.

    Vous pouvez saisir ce qui suit dans la zone de texte :

    • Description d'événement
    • Code d'événement complet, avec ou sans 0x.

  4. Cliquez sur Requête.

    Les événements correspondant aux critères de filtrage sont affichés.

Supprimer tous les événements système

Les événements système supprimés ne peuvent pas être restaurés. Soyez prudent lorsque vous supprimez des événements système.

  1. Choosissez Entretien > Alarme et SEL, et sélectionnez l'onglet Événements système.
  2. Cliquez sur dans le coin supérieur droit de la page.

    Tous les événements système seront effacés.

Télécharger les événements système

  1. Choosissez Entretien > Alarme et SEL, et sélectionnez l'onglet Événements système.
  2. Cliquez sur dans le coin supérieur droit de la page.

    Le fichier téléchargé est automatiquement enregistré dans le répertoire par défaut sur le PC local.
