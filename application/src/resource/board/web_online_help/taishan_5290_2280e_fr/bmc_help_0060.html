Infos capteur

Identifie un module logique ou une entité physique qui surveille les indicateurs du serveur.

Valeur actuelle

Valeur actuelle d'un indicateur.

-- : Le capteur n'a pas collecté de données valides.

Statut

Statut d'un capteur de seuil.

  • OK : Le capteur fonctionne correctement.
  • -- : Le statut du capteur est inconnu.
  • NC : Le capteur détecte une alarme mineure.
  • CR : Le capteur détecte une alarme majeure.
  • NR : Le capteur détecte une alarme critique.
