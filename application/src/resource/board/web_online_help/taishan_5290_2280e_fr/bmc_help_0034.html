Support virtuel

Présentation de la fonction

Vous pouvez utiliser le service VMM (Virtual Media Manager) pour monter un lecteur de CD ou de DVD virtuel ou un lecteur de disquette virtuel sur un serveur.

Sur la page Support virtuel , vous pouvez consulter et configurer les paramètres VMM.

Description du paramètre

Tableau 1 Support virtuel

Paramètre

Description

VMM

Statut du service VMM.

Il est activé par défaut.

Port

Numéro de port pour le service VMM.

Valeur : 1 à 65535

Valeur par défaut : 8208

REMARQUE:

Pour des raisons de sécurité, le navigateur désactive certains ports qui ne sont pas utilisés pour la navigation sur Internet. Si ces ports sont utilisés, le média virtuel de la console distante intégrée HTML5 ne sera pas disponible.

Chiffrement

Fonction de chiffrement des données de support virtuelles avant leur transmission.

Si cette fonction est activée, les données de support virtuelles sont cryptées à l'aide de l'algorithme AES128 avant d'être transmises entre le serveur et le client.

Par défaut, le chiffrement de support virtuel est désactivé. Pour des raisons de sécurité, activez cette fonction.

REMARQUE:

Le cryptage de communication n'est applicable qu'à la console à distance Java. La console à distance HTML5 utilise TLS pour le cryptage et ne repose pas sur la page actuelle pour le cryptage.

Nombre maximum de sessions

Nombre maximum d'utilisateurs simultanés autorisés à utiliser le support virtuel (lecteur DVD-ROM virtuel ou lecteur de disquettes) de la Console virtuelle à distance. Ce paramètre a une valeur fixe de 1.

Sessions actives

Nombre d'utilisateurs qui utilisent le support virtuel (lecteur de DVD-ROM virtuel ou lecteur de disquettes) de la console virtuelle à distance.
