BIOS配置

功能介绍

通过使用“BIOS配置”界面的功能，您可以设置操作系统第一选择从哪种设备进行启动。

参数说明

表1 启动项配置

参数

描述

优先引导介质
  • 硬盘：表示强制从硬盘启动系统。
  • 光驱：表示强制从CD/DVD启动系统。
  • 软驱/可拔插移动设备：表示强制从软驱或可拔插移动设备启动系统。
  • PXE：表示强制从预启动执行环境（PXE，Pre-boot Execution Environment）启动系统。
  • BIOS设置：表示服务器启动后直接进入BIOS菜单中。
  • 未配置：表示不设置第一启动盘，按BIOS中设置的方式启动操作系统。
  • 单次有效：优先引导介质的设置仅在下一次重启时生效，重启完成后，优先引导介质自动恢复为“未配置”。
  • 永久有效：优先引导介质的设置永久有效。

默认为“未配置”和“单次有效”。

启动顺序

“优先引导介质”为“未设置”时，按照“启动顺序”中的启动方式启动OS系统。

单击表示上移，单击表示下移。

说明：
  • 在BIOS侧，设置启动顺序后立即生效。重启OS将触发BMC启动顺序与BIOS侧启动顺序同步。
  • 在BMC侧，设置启动顺序后重启OS生效。重启OS将触发BIOS启动顺序与BMC侧启动顺序同步。

设置系统启动项

  1. 在“系统启动项”页签中，根据表1提供的参数信息，设置操作系统的第一启动盘。
  2. 单击“保存”。

    显示“保存成功”表示设置成功。
