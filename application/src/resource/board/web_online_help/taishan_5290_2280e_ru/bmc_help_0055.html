ПРИМЕЧАНИЕ:

В данной папке содержатся файлы из каталога /opt/pme/save. В файле *.md5 содержится код проверки целостности. В файле *.sha256 содержится код проверки целостности. Файл *.bak – файл резервной копии. Файл *.tar.gz – сжатый в пакет файл. Файл per_*.ini – файл постоянной конфигурации. Файл *sel.* – файл журнала системных событий. В данную папку не входят промежуточные файлы, создаваемые в процессе работы программы, и файлы, у которых нет проблем с информационной безопасностью.
