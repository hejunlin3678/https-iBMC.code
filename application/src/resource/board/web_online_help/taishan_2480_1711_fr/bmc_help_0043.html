Gestion de la licence

Présentation de la fonction

Sur la page Gestion de la licence, vous pouvez utiliser les fonctionnalités avancées d'BMC via une autorisation de licence.

Les utilisateurs peuvent utiliser les fonctionnalités avancées d'BMC pendant la période de validité de la licence. Sans la licence, seules les fonctionnalités par défaut de l'édition de veille BMC peuvent être utilisées.

L'édition avancée d'BMC fournit les fonctionnalités suivantes :

  • Déployez le système d'exploitation via l'interface Redfish.
  • Collectez les données originales de diagnostics intelligents via l'interface Redfish.
  • Activez les moteurs d'accélération Kunpeng, y compris le moteur de sécurité matérielle (SEC), le moteur RSA hautes performances (HPRE), le moteur RAID DIF (RDE) et le moteur ZIP.
  • Activez les mode de verrouillage du système.

Vous pouvez exporter une licence et la sauvegarder.

Révoquer

Révoquez une licence.

Lorsqu'une licence expire et entre dans la période de grâce, vous pouvez obtenir le code de révocation à partir de cette page. Par exemple, avant de remplacer un composant, vous devez cliquer sur le bouton Révoquer pour obtenir un code de révocation. Vous pouvez utiliser le code de révocation pour demander une nouvelle licence et installer la nouvelle licence sur le nouveau composant.

REMARQUE:

N'installez pas de licence invalidée.

État de la licence

L'état de la licence peut être l'un des suivants :

  • Normal : Une licence commerciale a été installée, la licence n'est pas expirée et toutes les fonctionnalités autorisées sont en état normal.
  • Mise en service : Une licence de mise en service a été installée, la licence n'a pas expiré et toutes les fonctions autorisées sont en état normal.
  • Période de grâce : Une licence commerciale ou de mise en service a été installée, la licence a expiré et pendant la période de grâce, toutes les fonctionnalités autorisées ne seront pas utilisées après l'expiration de cette période.
  • Par défaut : Une licence commerciale ou de mise en service a été installée et sa période de grâce a expiré.

Code de révocation

Un code généré après le bouton Révoquer est cliqué pour une licence installée. Vous pouvez utiliser ce code pour demander une nouvelle licence.

Informations sur la licence

Les informations de licence comprennent les éléments suivants :

  • Numéro de série de la licence
  • Type de licence
  • Les deux types de licence suivants sont disponibles :
    • Commercial : Une licence formelle délivrée pour un client sur la base du contrat. Toutes les fonctionnalités autorisées ont la même période de validité. La période de validité peut être Permanente ou une heure spécifique. Lorsque la licence expire, elle entre dans la période de grâce, qui est de 60 jours.
    • Essai : Une licence temporaire utilisée à titre d'essai pour utiliser de nouvelles fonctionnalités, la mise en service du système ou l'exposition de marques. La période de validité varie en fonction de la situation réelle. Lorsque la licence expire, elle entre dans la période de grâce, qui est de 60 jours.
  • Expiration : Date de fin de la licence. Il peut être Permanent ou à une heure précise.
  • Niveau de licence
    • Édition standard (par défaut) : Licence pour les fonctionnalités fournies par l'édition standard d'BMC. Les utilisateurs n'ont pas besoin d'acheter la licence.
    • Édition avancée : Licence pour les fonctionnalités fournies par l'édition BMC avancée. Les utilisateurs ne peuvent utiliser ces fonctionnalités qu'après l'achat d'une licence.
  • Période de grâce de la licence : Nombre de jours pendant lesquels le BMC peut encore être utilisé après l'expiration de la licence. Il a une valeur fixe de 60.

Fonctionnalités avancées

Affiche les informations sur les fonctionnalités avancées de la licence.

Les informations incluent le numéro, la fonctionnalité, l'état de la fonctionnalité, l'état actuel de la fonctionnalité et l'expiration de chaque fonctionnalité.
