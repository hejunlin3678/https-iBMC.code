Parameter

Description

Basic information

Provides the following information about the memory:

  • Maximum and actual number of DIMMs
  • Name, manufacturer, capacity, configured speed, maximum speed, type, and location of each DIMM

Details

Click to the left of the DIMM name to display the following information:

Part number, BOM code, serial number, bit width, number of ranks, minimum voltage, and type details
