Fan and Heat Dissipation

Function Description

On the Fan and Heat Dissipation page, you can:

  • View the historical data of the server air inlet temperature.
  • View and configure the fan adjustment settings.

    If the fan adjustment mode is Manual, all the settings on the Smart Cooling page will take effect only when the fan adjustment mode changes to Auto.

  • View information about the fans installed in the server.

Parameter Description

Table 1 Air Inlet Temperature

Parameter

Description

Inlet Temperature

Air inlet temperature data sampled every 10 minutes within the last one week.

Current Value

The latest temperature detected by the air inlet sensor.

Clear

Click Clear to clear all data.

Refresh

Click Refresh to update the current status to the statistics.
Table 2 Fan information

Parameter

Description

Basic information

Basic information about the fan modules of the server, including the slot number, name, model, speed, speed ratio and part number of each fan module.

Real-Time Refresh

Enables or disables the real-time refreshing of fan information.

Once enabled, the fan speed and speed ratio are automatically refreshed every second.

It is disabled by default.
Table 3 Smart Cooling Settings

Parameter

Description

Energy saving mode

It is the default mode for the air cooled system.

Adjusts fans to the system load for better heat dissipation and power savings.

Low noise mode

Minimizes fan speed to dissipate heat quietly.

High performance mode

Maximizes fan speed for optimal component cooling and system running.

Custom mode
Provides interfaces for users to customize the following parameters. Optional value ranges will be given as references based on the server load and heat dissipation situation.
  • CPU Target Temperature
  • Fan Speed
NOTE:
  • The fan speed corresponding to a higher temperature range must be greater than the fan speed corresponding to a lower temperature range.
  • The parameters available in the custom mode vary depending on the server model.
  • During CPU replacement, if the highest operating temperature of the new CPU is lower than the current value of CPU Target Temperature, the BMC automatically changes the value of CPU Target Temperature to the maximum temperature allowed by the new CPU.

Setting the Smart Cooling Mode

For example, to set the smart cooling mode to Custom mode, perform the following steps:

If the custom mode is not set properly, insufficient heat dissipation may be caused.

  1. Click Settings.
  2. Select Custom mode.
  3. Set CPU Target Temperature based on the value range displayed.
  4. Set the fan speeds corresponding to the inlet temperature ranges.
  5. Click Save.

    If "Operation successful" is displayed, the smart cooling mode is set successfully.
