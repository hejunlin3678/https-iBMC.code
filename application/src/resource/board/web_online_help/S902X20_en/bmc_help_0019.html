Video & Screenshot

Function Description

In the Video Playback area, you can:

  • Enable or disable the video recording function.

    When this function is enabled, BMC automatically records CPU error videos, shutdown videos, and restart videos.

  • Play a video file of the server stored on the local PC.
  • Play a video that was automatically recorded on the server.
  • Capture a picture during the playback of a video.
    • The video file must be in *.rep format.
    • The captured picture must be in the *.jpg format.
    • After the video recording function is enabled, sensitive service information may be captured during video recording.

In the Screenshot area, you can:

  • Enable or disable the last crash screen function.

    The last crash screen function allows automatic capture of the screenshot of the server screen just before the server is restarted or powered off.

  • Capture a screenshot of the server desktop in real time.

    The last screenshot function is enabled by default. After this function is enabled, sensitive service information may be captured in the screenshot.

    Play a video file from a specified point. (The slider indicates the playback progress.)

    Loop a video file.

    This function is available only for local video files.

Capture a screenshot.
  1. Click in the Capture Screenshot area.
