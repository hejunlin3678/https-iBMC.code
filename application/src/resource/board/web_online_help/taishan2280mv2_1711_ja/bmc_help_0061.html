パラメータの説明

表 1 その他

パラメータ

説明

HDDバックプレーン

ドライブバックプレーン情報を提供します。

  • ドライブバックプレーンの拡張可能な最大搭載数と現在の搭載数
  • 各ドライブバックプレーンの名前、位置、メーカー、番号、種類、PCBバージョン、CPLDバージョン、ボードID、部品番号、シリアル番号

I/Oボード
I/Oボード情報を提供します（I/Oボードをサポートしているサーバーのみ）。
  • I/Oボードの最大搭載数と現在の搭載数。
  • I/Oボードの名前、ロケーション、製造メーカー、種類、ファームウェアバージョン、PCBバージョン、CPLDバージョン、ボードID、電源、BOMコード、およびシリアル番号。

PCIeアダプタ

名前、説明、スロット、PCBバージョン、ボードID、などのPCIeアダプタ情報ーを提供します。

PCIeカード

PCIeカード情報を提供します。

  • PCIeカードの最大搭載数と現在の搭載数。
  • PCIeカードの詳細、ロケーション、製造メーカー、スロット番号、ベンダーID、デバイスID、部品番号、サブベンダーID、サブデバイスID、および接続先。
  • [詳細]の下のをクリックして、PCIeカードの詳細を表示します。
  • SDI 5.0カードの場合、[診断中断]をクリックしてマスク不可能割り込み（NMI）をトリガーできます。
    NOTE:

    中断情報が診断用にオペレーティングシステムに送信されるため、オペレーティングシステムがクラッシュし、データの損失や破損が発生する可能性があります。

RAIDカード

RAIDコントローラカード情報を提供します。

  • RAIDコントローラカードの最大搭載数と現在の搭載数。
  • RAIDコントローラカードの名前、ロケーション、製造メーカー、番号、種類、PCBバージョン、CPLDバージョン、ボードID、関連リソース、BOMコード、およびシリアル番号。

Riserカード

Riserカード情報を提供します。

  • Riserカードの最大搭載数と現在の搭載数
  • 各Riserカードの名前、メーカー、スロット、種類、PCBバージョン、ボードID、部品番号、シリアル番号

セキュリティモジュール

セキュリティモジュール情報を提供します。

  • セキュリティモジュールの最大搭載数と現在の搭載数。
  • セキュリティモジュールのプロトコルタイプ、プロトコルバージョン、製造メーカー、製造メーカーバージョン、およびセルフチェック状態。
