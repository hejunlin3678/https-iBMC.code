Smart Provisioning管理

機能説明

Smart Provisioningはインテリジェントなサーバー展開ツールです。

Smart Provisioning管理画面で、次の操作を実行することができます。
  • 機器情報収集機能を構成します。
  • Smart ProvisioningからOSを起動する機能を構成します。

パラメータの説明

表 1 Smart Provisioning管理

パラメータ

説明

サーバー情報の収集

この機能を有効にした場合、PSUの電源投入時にインバンドシステムが初めて起動した際、インバンドシステムがSmarting Provisioningにアクセスしてサーバー情報を収集します。その後、サーバーは[BIOS構成]の[優先起動媒体]および[起動順位]に従って起動します。

デフォルトでは無効になっています。

SPからOSを起動

この機能を有効にすると、サーバーを再起動する場合に、インバンドシステムは直接Smart Provisioningに入ります。

デフォルトでは無効になっています。

NOTE:

この設定は、次回の再起動時にのみ有効になります。システムがSmart Provisioningに入ると、この機能は自動的に無効になります。
